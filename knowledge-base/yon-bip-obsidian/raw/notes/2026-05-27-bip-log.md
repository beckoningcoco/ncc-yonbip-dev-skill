# 旗舰版日志记录

> 来源：yonyou-bip-dev/references/bip_log.md

## 客开代码中记录日志

```java
Logger log = LoggerFactory.getLogger(PatSnapServiceImpl.class);
log.error("sftp智慧芽文件已下载完成，准备处理！");
```

## 日志查看位置

1. 登录技术中台
2. 微服务管理节点，找到客开的微服务
3. 切换实例管理，找到 YMS 日志下载
4. 找到 `/微服务编码/yms-application.log` 文件

## 日志级别

- **推荐使用 `log.error()`**：BIP 系统默认记录日志级别为 error，低于 error 级别会被忽略
- **修改日志级别**：技术中台 → 配置管理 → 选择环境 → 客开微服务 → 日志页签 → 修改日志级别 → 单服务配置发布 → 重启微服务
