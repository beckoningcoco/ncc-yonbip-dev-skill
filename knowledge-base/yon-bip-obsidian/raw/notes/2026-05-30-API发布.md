## 业务场景

原厂单据调用自建单据进行新增、修改、删除操作，因为跨领域。所以必须将API脚本发布为接口，原厂才能调用；

三方系统需要调用自建单据进行CRUD操作，可以通过该方式将API脚本发布为接口，供三方系统调用。通过该文档会了解API发布的基本步骤。

## 创建API脚本

1. 先创建一个API脚本

![](https://ncc-pub-ublim.oss-cn-beijing.aliyuncs.com/help/photo/ce5e08e4-1034-438e-87b7-10aa42e13096.png)

![](https://ncc-pub-ublim.oss-cn-beijing.aliyuncs.com/help/photo/3b8babe2-8ee6-45b1-8940-92f10c8a1395.png)

内容：

```
|let AbstractAPIHandler = require('AbstractAPIHandler');

class MyAPIHandler extends AbstractAPIHandler {

execute(request) {

var sql = \\`select new1,new2 from AT20669A0A1CA00005.AT20669A0A1CA00005.k70734\_card\_demo where id=${request.id}\\`;

var res = ObjectStore.queryByYonQL(sql);

return { res };

}

}

exports({ entryPoint: MyAPIHandler });|
```

#常见问题

## 问题1：如何取消发布？

答：点击“取消发布”按钮，可以选择取消发布。

## 问题2：如何重新发布？

答：点击“重新发布”按钮，可以选择重新发布。

## 问题3：如何查看发布历史？

答：点击“发布历史”按钮，可以查看发布历史。

## 问题4：body参数根类型可以选择什么类型？

答：可以选择对象类型和数组类型。

## 问题5：接口超时时间怎么设置？

答：请求超时时间，必须在1~30秒之间，默认5秒。可以在“后端配置”页签设置。
