## 参照

## 1 参照简介

参照，即对其它实体信息的参考和对照。

**YonBIP高级版参照的实质：** 前端 JS：通过配置属性控制参照的界面显示。 后端 Java：拼接查询参照数据的具体SQL。

## 2 参照分类

参照分为以下三种，表型参照、树型参照和树表型参照。

**树形参照**

后端 Java 类需继承 DefaultTreeRefAction 类，前端 JS 配置中需加入 rootNode 和 TreeConfig 的配置一次请求。

![](https://community.yonyou.com/datadict/bipbook/assets/0002.sfF7KFqV.png)

**表型参照**

后端 Java 类需继承 DefaultGridRefAction 类，前端 JS 配置中需加入 ColumnConfig 的配置一次请求。

![](https://community.yonyou.com/datadict/bipbook/assets/0001.B8Oajvym.png)

**树表型参照**

实际上是树形参照和表形参照的集合。两次请求，一次请求走树形参照 action，一次请求走表形参照 action。

![](https://community.yonyou.com/datadict/bipbook/assets/0003.EtSipv6d.png)

继承了对应参照类型基类后，一般只需重写4个方法和构造方法。

1. 构造方法中设置管控模式，数据权限，停启用相关参数，后续在DefaultRefSqlbuilder类中会根据这些参数拼接SQL条件。
2. getRefMeta()方法中设置RefMeta对象各属性对应的String值，即sql语句中我们需要查询的字段，以及设置要查询的表名。
3. getExtraSql()方法会拼接Sql语句中where 后的条件语句。
4. getExtralSqlParameter()设置where后条件语句中占位符对应的真实数据（为了防止Sql注入）
5. getOrderSql()方法设置sql语句中的order by内容。

### 6.2.2 设计思路

- BaseRefer 是所有参照的基础，实现了除参照按钮以外的所有功能。因为点击参照按钮的操作有两类交互，故以此作为界限，划分出 BaseRefer。
- PopRefer 是点击参照按钮弹出模态框的一类按照，继承 BaseRefer，实现了父类的 show 方法和renderPopover 方法。
- DropDownRefer 是点击参照按钮弹出下拉列表的一类参照，继承 BaseRefer，实现了父类的 show 方法和renderDropDown 方法。
- 继承实际上也可帮我们做到组件的解耦，同时也给其他开发者提供了扩展组件的能力（复写类实例/原型方法）

## 6.3 自定义参照

我们实现自定义参照 ，尽量从 PopRefer 和 DrapDownRefer 继承，尽量不要从 BaseRefer 继承。如果继承 BaseRefer，需要自己实现 render 方法，会比较繁琐，而且一般情况下，我们从 PopRefer 继承就可以了，DrapDownRefer 是下拉的参照方式，下拉的数据格式比较固定，一般是列表结构的数据，不涉及交互方式的问题，主要的问题是弹出的参照问题，所以基本上通过继承 PopRefer。

### 6.4.1 实现自定义参照类

首先从继承PopRefer，实现自己的参照类。

```javascript
javascriptimport React, { Component } from 'react';
import {high} from 'nc-lightapp-front';
const { PopRefer, MultiLangWrapper } = high.Refer; // 引入PopRefer类

/**
 * 自定义参照
 */
class CustomRefer extends PopRefer { // 继承PopRefer类
    constructor(props){
        super(props)
    }
}

/***
 * 多语转换包装器
 *  把配置中的多语转换成相应的语种文字
 */
const AddressRefWrapper = MultiLangWrapper(CustomRefer);

/**
 * 抛出参照对象
 * @param {} props
 */
export default function (props = {}) {
    var conf = {
        multiLang: {
            domainName: 'uapbd',//领域编码
            currentLocale: 'zh-CN',//当前语种
            moduleId: 'refer_uapbd',//领域内public/lang/standard/simpchn/下的json文件名
        },
        refType: 'grid',//参照类型 grid/tree/gridTree
        refName: 'refer-000554',//参照名称的多语码
        placeholder: 'refer-000554',//默认提示多语码
        refCode: 'uapbd.pubinfo.AddressRef',//refCode 一般是：领域编码.模块编码.参照名
        queryGridUrl: '/nccloud/uapbd/ref/AddressRef.do',//grid类型为queryGridUrl,tree类型为queryTreeUrl
        columnConfig: [{ name: ['refer-000002', 'refer-000555'], code: ['refcode', 'refname'] }],
        isMultiSelectedEnabled: false,//是否多选
        showHistory: false,//鼠标聚焦参照输入框时显示历史数据
        popWindowClassName: 'ncc-address-refer-zIndex',//自定义参照样式
    };
    return <AddressRefWrapper {...conf} {...props} />
}
```

```javascript
javascript/**
 * 复写打开参照的方法
 * 由于是档案型参照，所以在有主键是需要加载对应的数据
 */
show = ()=>{
    let { disabled,form,value} = this.props;
    if (disabled) {
        return false;
    }
    !this.hasOwnProperty('prevOverFlow') && (this.prevOverFlow = document.body.style.overflow);
    let param = this.__getParam({});//获得所有参照

    if(!param.queryCondition || !param.queryCondition.pk){
        param.queryCondition.pk = value.value;
    }

    this.setState({
        isShow: true,
        isFirstShow: false,
        dropDownShow: false,
        referData:{ //打开时先清空数据
            a:{value:'',display:''},
            b:{value:'',display:''}
        }
    },()=>{
        //再加载数据
        this.loadData(param).then((data) => {
            this.setState({referData:data},()=>{});
        });
    });

}
/**
 * 加载数据
 */
loadData = async (param)=>{
    return await new Promise((resolve) => {
        this.setState(
            {
                loading: true
            },
            () => {
                let {queryGridUrl} = this.props;
                ajax({
                    url: queryGridUrl,
                    data: param,
                    loading: false,
                    success: (res) => {
                        resolve(res.data);
                    },
                    error: (e) => {
                        toast({ color: 'danger', content: e.message });
                        this.setState({
                            loading: false
                        });
                        throw new Error(e);
                    }
                });
            }
        );
    });
}
```

onSaveLinkman = (callback = ()=>{})=>{

 * 执行保存
 *******************************/
saveFun = (requestParam,callback = ()=>{})=>{
    let saveUrl= "你自己的保存的url";
    ajax({
        url:saveUrl,
        data:requestParam,
        success:(res)=>{
            if(res.success){
                //保存完成关闭弹出层
                let data = new Map();
                data.set(res.data.pk.value,res.data);
                this.setState({selectedValues:data},()=>{
                    callback();
                })
            }
        }
    })
}
