## 1 概述 [​](https://community.yonyou.com/datadict/bipbook/chapter1003/chapter1003.html#_1-%E6%A6%82%E8%BF%B0)

​ 在日常开发过程中，有一些功能需要初始数据，才能正常运行，这个时候，在做安装盘的过程中，就需要将这部分初始数据带上。一般情况下，导入初始数据的方式是在环境上执行插入 sql，来完成数据的初始化。idea/eclipse 中为我们提供了自动生成 sql 的插件功能。该功能通过 items.xml 配置文件来决定导出哪些数据表的哪些数据，进而自动生成 sql 脚本。

## 3.1 **items.xml** 文件 [​](https://community.yonyou.com/datadict/bipbook/chapter1003/chapter1003.html#_3-1-items-xml-%E6%96%87%E4%BB%B6)

一、文件位置

{项目}/{模块}/script/conf/items.xml

二、作用

1.配置导出预置脚本的表

2.配置导出数据的查询条件

三、文件内容

1.编码：文件内每个 item 标签的唯一标识

2.表名：idea/eclipse 根据表名，去查询相应的数据库表

3.查询条件：根据查询条件查出需要的初始数据，并生成插入数据的 sql 脚本

![](https://community.yonyou.com/datadict/bipbook/assets/3-1.BlLL0H7F.png)

## 3.2 **mapping.properties** **文件** [​](https://community.yonyou.com/datadict/bipbook/chapter1003/chapter1003.html#_3-2-mapping-properties-%E6%96%87%E4%BB%B6)

一、文件位置

{eclipse目录}/ .tables/common/mapping.properties

二、作用

用于配置生成预置脚本的目录

三、文件内容

1.键值对

2.键：表名

3.值：生成脚本文件在项目中的目录名

## 4 总结 [​](https://community.yonyou.com/datadict/bipbook/chapter1003/chapter1003.html#_4-%E6%80%BB%E7%BB%93)

1.items.xml 文件：数据库预置脚本的核心文件，一般情况下，只需要对该文进行修改即可，主要配置导预置脚本的表和表中满足条件的数据

2.mapping.properties 文件：用于配置生成文件的目录位置，由平台提供，一般不做配置

（1）如果配置，则导出目录为 script 目录下的一级目录

（2）如果没有配置，则导出目录为 script 目录下的 business 目录

3.{eclipse目录}/.tables/common/tablerule 中的 xml 配置文件：用于配置生成文件的主子表信息，由平台提供，一般不做配置
