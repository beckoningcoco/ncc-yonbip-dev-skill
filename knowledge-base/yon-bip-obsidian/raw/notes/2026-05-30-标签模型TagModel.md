## 如何获取TagModel（标签）模型

获取TagModel（标签）模型的方法有三种，需要根据页面中字段所在的位置来判断使用哪一种方式获取TagModel（标签）模型。

## 2.1、适用于获取查询区中的标签模型

```
viewModel.on('afterMount', function (data) {
    //1、获取查询区模型
    let filterViewModelInfo = viewModel.getCache('FilterViewModel');
    //2、查询区afterInit事件，必须放在页面模型的afterMount事件中才生效
filterViewModelInfo.on('afterInit', function(data){
    //3、获取标签预模型，cItemName：标签字段的字段别名
        let tagModelInfo = filterViewModelInfo.get('cItemName');
        //4、获取标签真实没模型
        let tagModelNewInfo = TagModelInfo.getFromModel();
    })
});
```

## 2.2、适用于获取表格中的标签模型

```
//1、获取表格模型
    let gridModelInfo = viewModel.getGridModel();
//2、获取表格行编辑模型
let editRowModelInfo = viewModel.getEditRowModel();
//3、获取标签模型，cItemName：标签字段的字段别名
    let tagModelInfo = editRowModelInfo.get('cItemName');
```

## 2.3、适用于获取单卡页的标签模型

//获取标签模型，cItemName：参照字段的字段别名

let tagModelInfo = viewModel.get('cItemName');

## TagModel（标签）模型预制方法

## 3.1、clear()

**方法：clear** ()

**使用说明：** TagModel的clear()方法，用于清除数据。

**参数：** 无

**示例代码：**

TagModel.clear();

## 3.2、setFilter(filter)

**方法：** setFilter(filter)

**使用说明：** TagModel的setFilter(filter)方法，用于设置标签过滤条件filter。

**示例代码：**

```
var myFilter = {"isExtend": true,simpleVOs:[]};
myFilter.simpleVOs.push({
field: 'name', //过滤字段
op: 'eq',       //比较符
value1: 'XXX'  //属性值
});
TagModel.setFilter(myFilter);
```

**使用说明：** TagModel的getFilter()方法，用于获取标签过滤条件。

**示例代码：**

TagModel.getFilter();

**使用说明：TagModel** 的validate()方法，用于校验模型选择的值，showValue

TagModel.validate();

## TagModel（参照）模型预制事件

## 4.1、beforeValueChange

****事件** ：** beforeValueChange

**使用说明：TagModel** 的beforeValueChange事件，是标签确定返回数据改变前事件, 返回true则继续进行value的改变，返回false则终止value 的改变。

**示例代码：**

//例：示例中data格式为{ value: value, oldValue: oldValue }

viewModel.get('staffNew\_name').on('beforeValueChange',function (data) {

return true;

});

## 4.2、afterValueChange

**事件：** afterValueChange

**使用说明：TagModel** 的afterValueChange事件，是标签确定返回数据改变后事件，允许对改变后的数据进行操作。

**示例代码：**

//例：示例中data格式为{ value: value, oldValue: oldValue }

viewModel.get('staffNew\_name').on('afterValueChange',function (data) {});

## 4.3、beforeBrowse

**事件：** beforeBrowse

**使用说明：TagModel** 的beforeBrowse事件，是表点击按钮弹出标签参照前事件，返回true则继续进行弹出参照操作，返回false 则终止弹出。

**示例代码：**

viewModel.get('staffNew\_name').on('beforeBrowse',function (data) {

return true;

});

## 4.4、afterBrowse

**事件：** afterBrowse

使用说明：TagModel的afterBrowse事件，是点击按钮弹出参照后事件。

**示例代码：**

viewModel.get('staffNew\_name').on('afterBrowse',function (rows) {});

## 问题1：表格里面的多选标签如何实现过滤

答：

1. 通过 viewModel.getGridModel() 或viewModel.get(“childrenField”) 获取表格模型；
2. 通过表格模型再获取表格编辑行模型即getEditRowModel()方法；
3. 然后再通过get(“字段别名”)获取表格中参照类型字段的指定列；
4. 监听这个列的’beforeBrowse’事件；在事件中即可设置复杂的过滤条件；
5. 详细示例见下方代码：
```
var girdModel = viewModel.getGridModel();  girdModel.getEditRowModel().get('ref1022_id').on('beforeBrowse', function (data) {
var condition = {
"isExtend": true,
simpleVOs: []
};
condition.simpleVOs.push({
field: 'new1',
op: 'eq',
value1: 11
});
this.setFilter(condition);
});
```

## 问题2：如何通过按钮弹出已有多选参照

答：

1. 通过 viewModel.get("参照CItemName") 获取参照模型；
2. 在按钮点击事件里面通过参照的browse()方法打开参照；
3. 详细示例见下方代码：
```
// 获取参照实例
const TagModel = viewModel.get("参照CItemName")
TagModel.setVisible(false);

// 按钮事件中弹出参照
viewmodel.get('按钮').on('click', function(data){
  TagModel.browse(true); // 参照弹出
})

// 参照值改变后事件
TagModel.on('afterValueChange',function(data){
  console.log(data.value); // 参照选择后的值
})
```

## 问题3：如何根据数据ID给多选参照赋默认值？

答：

1. 通过 viewModel.get("参照CItemName") 获取参照模型；
2. 通过cb.utils.triggerReferBrowse()方法赋值；
3. 详细示例见下方代码：

// item为参照TagModel

cb.utils.triggerReferBrowse(item, \[{ field:'id', op:'eq', value1:defaultValue }\]);

## 问题4：参照如何动态切换单选多选状态？

答：

1. 详细示例见下方代码:

// true 为多选 false 为单选

viewModel.get('sumSwitch').setMultiple(false/true);

## 问题5：如何设置标签颜色？

答：

1. 详细示例见下方代码：

// 获取参照实例后，可设置颜色值

const TagModel = viewModel.get("参照CItemName")

TagModel.setState(‘color’,’string’);
