# UI模板扩展

> 在 YonBIP 标准版环境中，基于原厂 UI 模板创建扩展模板，通过页面设计器修改页面布局和字段配置。

## 流程

1. 打开原厂应用 → **业务扩展** → 选择单据类型
2. 创建扩展模板（基于原厂模板复制）
3. 进入**页面设计器**，拖拽实体扩展字段到页面
4. **页面验证**：检查字段配置是否正确
5. **UI模板配置**：发布扩展模板

## 主实体 vs 子实体

| 类型 | 操作 | 说明 |
|------|------|------|
| 主实体扩展字段 | 在卡片页/列表页添加 | 拖拽到主表区域 |
| 子实体扩展字段 | 在子表区域添加 | 切换 Tab 到子表页签 |

## 页面设计器能力

- 字段拖拽布局
- 控件属性配置
- 事件绑定
- 样式调整

## 注意事项

- UI 模板扩展**必须基于已有模板**，不能凭空创建
- 扩展后需验证再发布
- 子实体字段需要在子表 Tab 页签里操作
