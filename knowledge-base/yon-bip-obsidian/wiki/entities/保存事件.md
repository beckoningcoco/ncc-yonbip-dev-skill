# 保存事件

> BIP 页面保存前后的完整事件处理方案。

## beforeSave — 保存前

```javascript
viewModel.on("beforeSave", function(args){
    let data = JSON.parse(args.data.data);
    // 校验逻辑
    if (!valid) { cb.utils.alert('错误', 'error'); return false; }
    args.data.data = JSON.stringify(data);
});
```

关键：`return false` 阻止保存，`return new cb.promise()` 支持异步确认。

## afterSave — 保存后

```javascript
viewModel.on('afterSave', function(data){
    if(!data.err){
        // data.res.id → 新保存的记录 ID
        // 可执行回调、刷新、回写等操作
    }
});
```

## 数据状态控制

```javascript
data._status = "Update";  // Insert→Update 强制更新
```
