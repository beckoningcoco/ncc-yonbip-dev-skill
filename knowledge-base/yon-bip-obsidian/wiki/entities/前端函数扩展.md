# 前端函数扩展

> YonBIP 标准版环境下，通过写 JavaScript 脚本实现页面控件的前端交互逻辑。

## 常见场景

- **值改变事件**：字段 A 的值变化 → 控制字段 B 的显示/可编辑
- **按钮点击事件**：自定义按钮点击后的逻辑
- **页面加载事件**：页面初始化时执行逻辑

## 开发步骤

1. UI设计器 → 选择目标控件
2. 添加事件（如 `afterValueChange`）
3. 在脚本编辑器中写 JS 代码
4. 保存 → 页面验证

## 调试方式

- 浏览器 F12 → Console 查看日志
- 在脚本中用 `console.log()` 输出调试信息

## 完整示例：客户档案"是否特殊客户"值改变事件

在"基础数据扩展"应用的"页面建模"页签，打开"客户档案卡片"编辑页面，选中页面顶级节点，在右侧"动作"页签 → 页面初始化 → 点击设置 → 选择前端函数：

```javascript
viewModel.on('customInit', function (data) {
    // 客户档案--页面初始化，默认置灰
    viewModel.get('extendNote').setDisabled(true);
});

viewModel.get('extendSpecial').on('afterValueChange', function (data) {
    var value = data.value;
    if (value == true) {
        viewModel.get('extendNote').setDisabled(false);
        viewModel.get('extendNote').setValue("");
    } else {
        viewModel.get('extendNote').setDisabled(true);
        viewModel.get('extendNote').setValue("");
    }
});
```

**关键 API**：
- `viewModel.on('customInit')` — 页面初始化完成
- `viewModel.get('字段编码').on('afterValueChange')` — 字段值改变事件
- `.setDisabled(true/false)` — 置灰/启用控件
- `.setValue("")` — 清空字段值

## 调试方式

- 浏览器 F12 → Console 查看日志
- 在脚本中用 `console.log()` 输出调试信息
- 或使用引擎命令 `yya` 在 Console 中调用：`yya.getAllData()` 查看页面数据
