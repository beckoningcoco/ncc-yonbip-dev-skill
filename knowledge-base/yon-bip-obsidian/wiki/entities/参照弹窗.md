# 参照弹窗

> BIP 参照弹窗的过滤、拦截、回填方案。

## 关键 API

| API | 说明 |
|-----|------|
| `this.setFilter(condition)` | 普通参照过滤 |
| `this.setTreeFilter(condition)` | 树形参照过滤 |
| `data.cancel = true` | 阻止弹窗打开 |
| `cb.utils.triggerReferBrowse(field, conditions)` | 程序化设置参照值 |

## 常用场景

- **权限过滤**：`{ field: 'id', op: 'in', value1: orgIds }`
- **状态过滤**：`{ field: 'enable', op: 'eq', value1: 1 }`
- **联动过滤**：主表字段值变化 → 子表参照动态过滤
- **前置校验**：参照打开前检查必要条件，不满足则 `data.cancel = true`
