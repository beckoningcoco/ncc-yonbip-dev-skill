# 可视化应用扩展

> [[低代码开发]] 的子能力之一：打开原厂应用进行个性化扩展，使其适配企业特殊需求。

## 官方定义

打开原厂应用，进行数据扩展、页面扩展、业务规则扩展，使原厂应用适应个性化需求；支持原厂应用扩展的设计能力。

## 扩展维度

| 扩展类型 | 说明 |
|---------|------|
| 数据扩展 | 字段扩展、子实体扩展 |
| 页面扩展 | 容器/控件组合设计、多页签、表格、按钮 |
| 业务规则扩展 | 页面规则、公式、前后端脚本 |
| 动作扩展 | 扩展原厂单据动作（save/delete/submit/audit等） |
| 流程扩展 | 审批流流程扩展、业务流推拉单回写 |

## 具体实现

[[可视化应用扩展]] 的具体技术实现是 [[原厂单据扩展开发]]。

## 与其他概念的关系

- 作用于 [[原厂单据]]（而非 [[自建单据]]）
- 通过 [[元数据]] 驱动实现字段扩展
- [[特征]] 机制提供动态属性扩展能力
- 扩展字段支持 OpenAPI 传参
