# 对照表模式

> BIP 客开中外部系统编码与 BIP 内部 ID 之间的映射翻译机制。是所有外部系统集成的基础设施。

## 数据模型

| 字段 | 说明 |
|------|------|
| 对比字段 | 外部系统编码 |
| 目标字段 | BIP 内部 ID/编码 |
| 来源系统 | 佳碟 / 巨益 / 每刻 |
| 生效日期 | 对照关系有效期开始 |
| 失效日期 | 对照关系有效期结束 |

## 唯一性校验

来源系统 + 对照类型 + 外部编码 + 有效期不能重叠。

通过 `BillPlugin.beforeSave()` 实现校验。

## 典型对照表

| 对照表 | 用途 |
|--------|------|
| 销售对照表 | 外部客户→BIP 客户/组织/部门/仓库 |
| 场景对照表 | 外部单据类型→BIP 交易类型 |
| 部门对照表 | 外部部门编码→BIP 部门 |
| 仓库对照表 | 外部仓库编码→BIP 仓库 |
| 每刻组织对照表 | 每刻组织编码→BIP 财务组织 |
