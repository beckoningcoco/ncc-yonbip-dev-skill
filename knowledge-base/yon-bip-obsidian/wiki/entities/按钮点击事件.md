# 按钮点击事件

> BIP 页面按钮交互的完整方案。

## 事件绑定

```javascript
viewModel.get('btnCode').on('click', fn);        // 点击时
viewModel.get('btnCode').on('beforeclick', fn);  // 点击前（返回 false 阻止）
```

## 按钮类型

| 类型 | 典型实现 |
|------|----------|
| 新增/编辑 | `cb.loader.runCommandLine('bill', request, vm)` |
| 保存并新增 | `viewModel.execute('save').then(() => execute('add'))` |
| 批量操作 | 遍历 GridModel rows → `invokeFunction` |
| 防重复点击 | `button.setDisabled(true)` → 完成后恢复 |

## 模态框按钮

```javascript
// 确定并回写
parentViewModel.getGridModel().appendRow(row);
viewModel.communication({ type: 'modal', payload: { data: false } });
```
