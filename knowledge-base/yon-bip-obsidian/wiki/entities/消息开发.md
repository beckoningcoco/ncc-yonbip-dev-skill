# 消息开发

> iuap 消息平台，支持通过 RPC 和 REST 接口发送站内消息、短信、邮件，并整合消息模板、消息策略、预警和流程通知。

## 核心架构

| 层级 | 说明 |
|------|------|
| **消息通道** | 站内信、短信、邮件、第三方APP推送 |
| **消息模板** | 预置系统级模板 + 租户级自定义模板 |
| **业务消息配置** | 将消息模板绑定到具体业务事件 |
| **消息策略** | 黑名单、白名单、限流策略 |

## 开发流程

### 1. 接入准备

注册应用 → 预置模板类型 → 设置消息模板 → 业务消息配置

### 2. 发送消息（RPC）

```java
// 发送模板消息给友户通用户
MessageCenterClient.sendTemplateMsg(userId, templateCode, params);

// 发送普通消息给组织或角色
MessageCenterClient.sendMsgToOrg(orgId, content);
MessageCenterClient.sendMsgToRole(roleId, content);
```

### 3. 发送消息（REST）

```
POST /message-center/api/send
POST /message-center/api/sendTemplate
```

## 消息通道设置

- **模板类型**：短信、邮件、站内信、微信、钉钉
- **模板设置**：系统级模板（所有租户共用）+ 租户级模板
- **业务配置**：将模板绑定到业务事件（如"订单创建"→ 发送"下单成功通知"）
- **预警配置**：库存预警、到期提醒等自动触发
- **流程消息**：审批任务到达、流程完成等通知

## 接入场景

| 场景 | 方式 |
|------|------|
| MDD 服务接入 | RPC 调用消息平台 SDK |
| 非 MDD/外部系统 | REST API 调用 |
| 第三方 APP 推送 | 扩展消息通道代理 |
| 混合云 | REST API 跨云调用 |
