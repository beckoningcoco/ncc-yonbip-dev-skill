# 状态控制

> BIP 页面中字段、按钮、表格行/列的动态状态控制。

## 控制层级

| 层级 | API | 作用对象 |
|------|-----|----------|
| 页面级 | `field.setVisible/setDisabled/setState` | 单个字段 |
| 表格行级 | `gridModel.setActionsState(states)` | 每行的操作按钮 |
| 表格行级 | `gridModel.setRowState(index, 'readOnly', bool)` | 整行只读 |
| 表格列级 | `gridModel.setColumnState(col, 'formatter', fn)` | 列格式化（颜色等） |
| 表格级 | `gridModel.setState('showRowNo', true)` | 显示行号 |

## 触发时机

- `modeChange` — 页面模式切换
- `afterLoadData` — 数据加载后
- `afterStateRule` — 状态规则执行后
- `afterSetDataSource` — 表格数据源设置后
