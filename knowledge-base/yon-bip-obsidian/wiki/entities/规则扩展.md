# 规则扩展

> 专业版环境下，通过 Java 开发 Rule 规则类，实现复杂的业务逻辑校验和数据处理，支持远程调用。

## 两种模式

| 模式 | 说明 | 回滚机制 |
|------|------|---------|
| **普通远程规则** | 单个 Rule 类，实现 `execute()` 方法 | 无 |
| **分布式远程规则** | 多个 Rule 类协同，支持 `cancel()` 回滚 | `execute()` 失败 → 调用 `cancel()` |

## 开发步骤

1. 在 Developer 中创建 Rule 工程
2. 编写 Java 类，继承 `AbstractCommonRule`
3. 实现 `execute()` 方法（核心逻辑）
4. 分布式模式还需实现 `ISagaRule` 接口的 `cancel()` 方法
5. 工程流水线发布 → YDS 制盘 → 部署
6. 在 UI 保存按钮配置远程调用

## 配置步骤

1. 编辑页面 → 保存按钮 → 动作 → 单击 → 配置
2. 新增规则，输入 Rule 名称（如 `merchantCheckRule`）
3. 服务域填引擎编码（如 `kkfunfypd`）
4. 将规则拖到规则链最上面（优先执行）

## 完整示例 1：普通远程规则

```java
package com.yonyou.ucf.mdf.domain.rule;

import com.yonyou.cloud.utils.StringUtils;
import com.yonyou.iuap.ucf.common.exception.BusinessException;
import com.yonyou.ucf.mdd.common.model.rule.RuleExecuteResult;
import com.yonyou.ucf.mdd.ext.bill.rule.base.AbstractCommonRule;
import com.yonyou.ucf.mdd.ext.model.BillContext;
import org.imeta.orm.base.BizObject;
import org.springframework.stereotype.Component;
import org.springframework.util.CollectionUtils;

import java.util.List;
import java.util.Map;

/**
 * 客户档案保存规则校验
 */
@Component("merchantCheckRule")
public class MerchantCheckRule extends AbstractCommonRule {

    @Override
    public RuleExecuteResult execute(BillContext billContext, Map<String, Object> map)
            throws Exception {
        List<BizObject> bills = this.getBills(billContext, map);
        if (!CollectionUtils.isEmpty(bills)) {
            BizObject bill = bills.get(0);
            String code = bill.get("merchantAppliedDetail!searchcode");
            if (StringUtils.isEmpty(code)) {
                throw new BusinessException("助记码不能为空!");
            }
        }
        return new RuleExecuteResult();
    }
}
```

## 完整示例 2：分布式远程规则（支持回滚）

```java
// TestSagaUpRule — 上游规则
@Component("testSagaUpRulezxy")
public class TestSagaUpRule extends AbstractCommonRule implements ISagaRule {

    @Override
    public RuleExecuteResult execute(BillContext billContext, Map<String, Object> paramMap)
            throws Exception {
        RuleExecuteResult result = new RuleExecuteResult();
        List<Map<String, Object>> params = new ArrayList<>();
        return result;
    }

    @Override
    public RuleExecuteResult cancel(BillContext billContext, Map<String, Object> paramMap)
            throws Exception {
        RuleExecuteResult result = new RuleExecuteResult();
        try {
            throw new BusinessException("down 规则报错后，会回滚执行 up 的 cancel 方法");
        } catch (BusinessException e) {
        }
        return result;
    }
}
```

```java
// TestSagaDownRule — 下游规则（故意报错测试回滚）
@Component("testSagaDownRulezxy")
public class TestSagaDownRule extends AbstractCommonRule implements ISagaRule {

    @Override
    public RuleExecuteResult execute(BillContext billContext, Map<String, Object> paramMap)
            throws Exception {
        logger.info("zxy saga down start-故意报错");
        RuleExecuteResult result = new RuleExecuteResult();
        List<Map<String, Object>> params = new ArrayList<>();
        SqlHelper.insert("insert tt", params);
        BizObject merchantBizObject = new BizObject();
        merchantBizObject.set("id", "2222");
        merchantBizObject.set("transType", "");
        MetaDaoHelper.update("dddd", merchantBizObject, "dd");
        MetaDaoHelper.insert("ddd", merchantBizObject);
        return result;
    }

    @Override
    public RuleExecuteResult cancel(BillContext billContext, Map<String, Object> paramMap)
            throws Exception {
        // Down 报错时，平台自动回滚，调用所有参与 Saga 规则的 cancel
        RuleExecuteResult result = new RuleExecuteResult();
        return result;
    }
}
```

**关键类说明**：
- `AbstractCommonRule` — 规则基类，必须继承
- `ISagaRule` — 分布式事务接口，需实现 `cancel()` 方法
- `BillContext` — 单据上下文
- `BizObject` — 业务对象
- `RuleExecuteResult` — 规则执行结果

## 远程调试

1. 流水线部署时勾选"启用调试端口"→ 平台自动生成 IP + 端口
2. Idea → Edit Configurations → 添加 Remote → 填 Host + Port
3. 启动远程调试（小蜘蛛图标）→ Rule 上打断点 → 页面操作触发

## 扩展字段取值

Rule 中获取扩展字段值：
```java
BizObject bill = bills.get(0);
String value = bill.get("extendFieldName");  // 直接用扩展字段的编码
```
