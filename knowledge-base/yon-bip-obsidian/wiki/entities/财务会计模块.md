# 财务会计模块

> YonBIP 5.0 财务云-财务会计，8 个子模块共 1475 页官方文档。

## 模块清单

| 模块 | 页数 | 核心功能 |
|------|------|---------|
| **总账** | 386 | 凭证管理、科目设置、期末结转、多账簿、合并报表 |
| **应收管理** | 201 | 应收单据、收款核销、坏账计提、账龄分析 |
| **应付管理** | 258 | 应付单据、付款核销、预付管理、供应商对账 |
| **存货核算** | 162 | 入库核算、出库核算、成本调整、存货盘点 |
| **固定资产** | 260 | 资产卡片、折旧计提、资产变动、资产减少、资产盘点 |
| **收入管理** | 149 | 收入确认、收入结算、收入分摊 |
| **业财多维分析** | 37 | 多维度财务数据分析、自助报表 |
| **收入稽核清分** | 91 | 收入数据的稽核和对账清分 |

## 开发相关

跨模块通用概念：
- **凭证**：总账核心单据，各模块业务单据通过会计平台生成凭证
- **会计平台**：业务单据 → 凭证的转换引擎
- **期末结转**：月末自动结转损益、计提折旧等
- **多账簿**：同一业务多会计核算体系
