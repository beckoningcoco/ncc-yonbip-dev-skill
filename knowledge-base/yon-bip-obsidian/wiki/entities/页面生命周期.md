# 页面生命周期

> BIP 页面从初始化到销毁的完整事件序列。

## 执行顺序

```
customInit → afterInit → afterInitCommonViewModel → afterMount
→ beforeLoadData → afterLoadData → modeChange
→ afterStateRule/afterRule
→ beforeSave → afterSave
→ afterAudit/afterUnaudit
```

## 核心事件速查

| 事件 | 用途 |
|------|------|
| `customInit` | 最早时机，注册监听器 |
| `afterMount` | DOM 完成，操作查询区 |
| `afterLoadData` | 数据加载完，控制界面状态 |
| `modeChange` | add/edit/browse 切换 |
| `afterStateRule` | 补充状态规则 |
| `afterAudit` | 审批后处理 |
