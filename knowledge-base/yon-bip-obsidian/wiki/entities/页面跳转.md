# 页面跳转

> BIP 中页面跳转的完整实现方式。

## 方式一：cb.loader.runCommandLine

```javascript
let request = {
    billtype: 'Voucher',       // Voucher(卡片) / VoucherList(列表)
    billno: '单据号',
    params: { mode: 'add', readOnly: false, id: '123', data: {...} }
};
cb.loader.runCommandLine('bill', request, viewModel);
```

## 方式二：模态框通信

- `viewModel.communication({ type: 'return', payload })` — 关闭并返回数据
- `viewModel.communication({ type: 'modal', payload: { data: false } })` — 关闭不返回

## 方式三：其他

- `window.open(url, '_blank')` — 外部系统
- `rootStore.pageStore.push({...})` — rootStore
- `jDiwork.openService(code, dt, param)` — 新标签页
