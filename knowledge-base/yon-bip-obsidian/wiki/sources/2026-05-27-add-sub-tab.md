# 如何新增子页签

> 扩展实体新增子页签的完整流程：扩展实体 → 手动建表 → 修改页面建模 → 修改 UI 模板。

## 步骤

1. **扩展实体**：在实体扩展中新增子页签的实体
2. **手动建表**：导出 SQL 建表语句（通常为 MySQL 格式），转换为达梦格式，在对应领域（如 `znzb`）中执行
3. **修改页面建模**：在应用构建中修改页面建模，增加子页签配置
4. **修改 UI 模板**：复制原厂模板，因为旧的模板没有新增页签；设置模板优先级调高
