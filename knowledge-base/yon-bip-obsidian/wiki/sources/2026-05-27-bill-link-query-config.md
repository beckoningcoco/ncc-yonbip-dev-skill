# 旗舰版单据联查通过联查配置关联两个单据

> 通过联查配置将两个没有上下游关系的单据（如收款单和应收发票）关联起来，在全局联查中可见。

## 配置步骤

1. 进入**联查配置**节点
2. 配置关联关系（下游单据号、所属应用等）

## 关键参数获取

- **下游单据号**：从业务对象或 UI 模板获取
- **所属应用**：从应用构建中或联查配置节点搜索单据查看预制配置中的应用编码
