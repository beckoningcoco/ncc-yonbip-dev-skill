# BIP 旗舰版日志记录

> 旗舰版客开代码中如何记录和查看日志。

## 核心观点

- 使用 `LoggerFactory.getLogger()` + `log.error()` 记录日志
- BIP 系统默认日志级别为 **error**，低于此级别会被忽略
- 日志文件位于 `/微服务编码/yms-application.log`，通过 YMS 日志下载获取
- 修改日志级别需在技术中台配置管理 → 日志页签 → 单服务配置发布 → 重启微服务
