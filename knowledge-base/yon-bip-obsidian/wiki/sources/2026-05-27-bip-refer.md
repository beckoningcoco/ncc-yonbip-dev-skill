# 20个BIP参照弹窗代码片段

> BIP 参照弹窗的过滤、确认、联动完整方案。

## 参照过滤

| 方法 | 适用 | 说明 |
|------|------|------|
| `this.setFilter(condition)` | 普通参照 | 列表型参照过滤 |
| `this.setTreeFilter(condition)` | 树形参照 | 树形参照过滤 |
| `data.cancel = true` | 通用 | 阻止参照弹窗打开 |

## beforeBrowse 过滤
```javascript
viewModel.get('field_name').on('beforeBrowse', function(data){
    this.setFilter({ isExtend: true, simpleVOs: [
        { field: 'enable', op: 'eq', value1: 1 }
    ]});
});
```

## 参照选择确认
```javascript
viewModel.get('field_name').on('afterReferOkClick', function(data){
    // data[0].id, data[0].name
});
```

## 程序化设置参照值
```javascript
cb.utils.triggerReferBrowse(field, [
    { field: 'id', op: 'eq', value1: id }
]);
```

## 前置校验（beforeBrowse 中）

if (!prerequisite) {
    cb.utils.alert('请先选择XX', 'error');
    data.cancel = true; return false;
}
