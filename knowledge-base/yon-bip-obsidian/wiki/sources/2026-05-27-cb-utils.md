# 后端函数工具类 cb.utils

> cb.utils 是 BIP 页面脚本中最常用的工具类，覆盖数组判断、对象拷贝、弹窗提示、ID 生成等功能。

## 常用方法速查

| 方法 | 功能 |
|------|------|
| `cb.utils.isArray(data)` | 判断是否为数组 |
| `cb.utils.extend(true, {}, obj)` | 深拷贝对象 |
| `cb.utils.isEmpty(value)` | 判空（null/空字符串/undefined） |
| `cb.utils.getNewId('prefix')` | 生成递增前缀 ID |
| `cb.utils.stringToJSON(str)` | JSON 字符串转对象 |
| `cb.utils.alert(msg, 'error')` | 弹窗提示（error/warning/info/success） |
| `cb.utils.confirm(msg, ok, cancel)` | 确认框提示 |
| `cb.utils.isEmptyObject(data)` | 判断是否为空对象 |
| `cb.utils.loadingControl.start()` / `end()` | 开启/关闭 loading |
| `cb.utils.queryString(url)` | URL 参数解析（get/set/has/del） |
| `cb.utils.getFullNum(num)` | 科学计数法转数字 |
| `cb.utils.removeEmplyAttr(obj)` | 删除对象中空值 |

## 同步调用后端

`cb.rest.invokeFunction("uri", params, callback, viewModel, {async: false})` 用于页面脚本同步调用后端 API 函数。
