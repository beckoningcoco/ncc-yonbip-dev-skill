# 旗舰版环境搭建

> V3 R6 以下用 IDEA，R6 及以上用 YDS 工具。

## 核心观点

- **V3 R6 以下**：使用 IDEA 搭建，需配置 Maven、数据源、启动参数、hosts
- **V3 R6 及以上**：使用 YDS 工具搭建
- 数据库默认使用达梦（DM），连接串格式 `jdbc:dm://host:port/DB_NAME`
- 启动参数关键项：`-Duse_local_conf=true`、`-Dlocal_conf_path=...`、`-Dmw_profiles_active=test`
- 需要配置 hosts 映射和维护 Maven 离线仓库

## 关键启动参数

| 参数 | 说明 |
|------|------|
| `-Dyms.logging.console=true` | 启用 YMS 控制台日志 |
| `-Denable_cloud_debug=true` | 启用云调试 |
| `-Duse_local_conf=true` | 使用本地配置 |
| `-Dlocal_conf_path=...` | 本地 YMS 配置文件路径 |
| `-Dmw_profiles_active=test` | 激活 test 环境配置 |
