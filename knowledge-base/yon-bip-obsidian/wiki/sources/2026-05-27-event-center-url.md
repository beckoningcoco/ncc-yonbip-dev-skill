# 事件中心地址

> 事件中心（事件监听注册）的访问地址模板。

## 地址模板

```
{域名}/iuap-apcom-event/ucf-wh/eventCenter/index.html#/event-listener-regist
```

将 `{域名}` 替换为实际的 BIP 环境域名即可访问事件监听注册页面。
