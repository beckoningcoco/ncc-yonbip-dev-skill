# 监控日志磁盘满导致API超时或日志不记录

> 监控日志磁盘满导致 API 超时和日志不记录的问题排查。

## 现象

1. 监控日志节点不记录日志
2. 调用原厂 API 耗时极长，批量调用时总有几条超时 60+ 秒

## 根因

监控日志服务磁盘满，API 写入日志时超时影响接口响应时间。

## 解决

- **临时方案**：清理监控日志磁盘旧数据
- **长期方案**：扩容磁盘空间

在技术中台查看监控日志服务状态，确认磁盘使用率。
