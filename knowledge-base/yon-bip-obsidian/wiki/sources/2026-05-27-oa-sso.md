# OA单点登录

> 实现泛微 OA（Weaver）与 YonBIP 的 SSO 集成。OA 使用 AES 加密用户凭证后重定向至 YonBIP SSO 接口，YonBIP 解密验证后生成登录 Token。

## 请求地址

```
GET /c-yonbip-hawk-asset/sso/index?thirdapp=weaver&usercode=<AES>&password=<AES>
```

## 加密规范

- 算法：AES/ECB/PKCS5Padding
- 密钥长度：128 位（16 字节）
- 输出：加密字节 → Base64 编码

## Controller 核心流程

1. 接收 OA 重定向请求，解密用户凭证
2. `UserCenter.createAccessToken` 验证身份
3. `UserCenter.getLoginTokenByAccessToken` 获取登录 Token
4. 重定向到 CAS 登录页（`/cas/login?token=...&service=...`）

使用 `RobotExecutors.runAs(tenantId, ...)` 以租户身份执行。
