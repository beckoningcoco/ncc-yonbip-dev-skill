# 在原厂模板新增了按钮如何同步已有的UI模板

> 在应用构建中扩展原厂模板新增按钮后，已有 UI 模板上不显示。

## 场景

事项分录卡片扩展新增按钮后，只有原厂模板有按钮，已建的 UI 模板没有。

## 原因

事项分录没有交易类型特性，无法通过交易类型同步原厂按钮到已有模板。

## 解决

需要通过截图中展示的其他途径同步 UI 模板（复制模板或重新配置）。
