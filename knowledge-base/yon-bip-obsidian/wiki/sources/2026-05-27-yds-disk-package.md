# YDS出盘制盘

> 在 YDS 环境中进行出盘制盘流程：Git 拉代码 → 产品包管理发布脚本 → 下载脚本 → 生成模板 → 勾选出盘 → 制盘打包。

## 流程步骤

1. **Git 拉取最新代码**
2. **产品包管理发布脚本**：对每个模块点"发布"，确保使用最新脚本
3. **下载脚本**：执行下载命令后工程多出脚本目录
4. **生成模板**：如制盘视图有旧模板先删除，重新生成
5. **勾选出盘项目**：出脚本勾选两个（项目+脚本），不出脚本只勾项目
6. **填写信息**：右边必填字段通常自动填充，未填充则关闭页签重新点击
7. **制盘**：点击"制盘"，生成压缩包即为最终补丁盘
