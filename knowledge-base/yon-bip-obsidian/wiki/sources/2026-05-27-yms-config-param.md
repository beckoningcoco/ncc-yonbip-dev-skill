# 后端函数获取YMS配置的参数

> 在后端函数中获取 YMS 配置的参数值。

## 代码

```java
// let url = AppYmsValue('domian.url');
```

使用 `AppYmsValue` 函数读取 YMS 配置中心（如 application.properties 或 YMS 控制台）中配置的参数值。
