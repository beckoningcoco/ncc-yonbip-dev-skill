# API 发布

> YonBIP 社区文档中心 — 将API脚本发布为对外接口。

## 核心内容

### 适用场景
- 原厂单据跨领域调用自建单据（必须发布）
- 三方系统调用自建单据CRUD操作

### 发布步骤
1. **创建API分组**：API发布节点→创建一/二/三级分组目录
2. **创建API脚本**：使用 `AbstractAPIHandler` 编写业务逻辑
3. **发布**：选中分组→新增→填写请求路径→添加参数→选择函数→勾选返回结果透传→保存→发布
4. **调试**：点击"调试"→输入参数→发起调试→查看响应

### 配置选项
- **超时时间**：1~30秒，默认5秒（后端配置页签设置）
- **限流控制** / **IP访问控制**
- 支持取消发布、重新发布、查看发布历史
