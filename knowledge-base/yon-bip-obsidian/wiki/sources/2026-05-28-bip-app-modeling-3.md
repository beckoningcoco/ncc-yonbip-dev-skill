## APP 数字化建模（3）：基础数据管理与系统管理

YonBIP V5.0.2507 APP 数字化建模培训课件第 3 部分（约 87 页），覆盖基础数据管理（分级管控/物料/供应商/客户）、系统管理（工作台/日志/预警调度）和工具集（导入导出/迁移/配置工作台）。

### 一、基础数据管理

BIP 主数据共 168 个档案节点，覆盖公共信息（国家/地区、行政区划、币种、计量单位等）、财务信息（会计主体、科目、费用项目、结算方式等）、物流信息（仓库、货位、物流公司、发运方式等）、工程数据（物料清单、工艺路线、工作中心等）。

#### 1. 基础数据分级管控

**分级管控本质**：明确数据在不同组织之间的管理权和使用权。

**四种数据类型**：
| 类型 | 示例 | 管控方式 |
|------|------|---------|
| 公共不带组织数据 | 币种、汇率 | 不管控，任何组织可见 |
| 普通带组织数据 | 税率、项目 | 按管控实现可见 |
| 树形带组织数据 | 物料分类、客户分类 | 按管控实现可见 |
| 带有业务属性数据 | 客户、供应商、物料 | 基本信息管控 + 业务信息属性管控 |

**管控规则**：全部组织分配 / 分配下级组织 / 自由分配 / 私有。核心档案还可定义业务信息字段的管控（使用组织可修改 / 不可修改），支持按数据条件配置多条分配规则。

#### 2. 物料档案

物料是系统中最核心的基础数据，贯穿研发→采购→生产→仓储→销售→核算全过程。

**物料管理流程**：确认管控策略 → 基础配置（编码规则/唯一性/分类/计量单位/特征）→ 定义物料模板（自由项/属性/SKU/选配特征）→ 物料创建（申请单/直接建档）→ 管理商品 SKU

**物料模板核心要素**：
- **自由项特征**：物料的动态特征。物料+自由项特征维度细化管理，支持 SKU 敏感、存量敏感
- **物料属性特征**：物料的静态属性。"特征→特征组→物料模板→物料档案"链路
- **SKU 属性特征**：描述 SKU 商品的静态属性，在 SKU 档案上维护
- **选配特征**：按单选配的派生物料模式参数化选配

**特征限制**：每种特征组类型下所有特征组扩展特征总数 ≤ 765 个。自定义档案+字符 ≤ 165 个、数值 ≤ 150 个、基础档案 ≤ 150 个、布尔 ≤ 150 个、日期 ≤ 150 个。

**应用价值**：统一编码与档案管理（消除信息孤岛）、分级管控与多样管控模式、特征提供随需而变的扩展能力、OpenAPI 开放集成。

#### 3. 供应商档案

**管理维度**：供应商分类、供应商申请 → 供应商档案（基本信息 + 资质信息 + 银行信息 + 联系人 + 地址 + 业务信息）

**管控模式**：
- **单组织**：自主创建和使用
- **多组织统一管控**：集团统一创建/维护，各分子公司共享使用，分子公司维护业务信息
- **多组织多级管控**：集团和有权限的组织都可创建，共享分配其他组织使用

**应用价值**：灵活分配共享范围、多组织支持、个性化模板、多管控模式、客商关系设置（内部供应商/客商关系/父子关系）、企业画像与风险订阅、混合云集成。

#### 4. 客户档案

**管理维度**：客户分类、销售区域、销售渠道、客户级别、客户行业

**核心能力**：客户申请单（新增准入/变更审核）、统一数据存储服务、客户全生命周期链路打通（CRM 客户→渠道客户→商家模型）、企业画像（工商资质核实）、字段级管控（全局字段/组织级字段）、混合云集成。

**业务流程**：潜在客户 → 正式客户（CRM）→ 渠道商家 → 渠道交易关系 → 渠道客户。支撑营销云（会员/定价/促销/返利/信用/B2B/B2C）、供应链云（报价/合同/销售订单/发票）、财务云（应收/收款/凭证）等多领域业务。

#### 5. 业务伙伴

代表现实世界的自然实体。解决"业务往来对象"在企业生态链中兼具多重角色的问题（一个业务伙伴可以是客户、供应商、内部组织、资金伙伴）。

**核心价值**：支持总账按统一对象查询账表（按业务伙伴作为辅助核算项）、支持客商合并统计。

### 二、系统管理

#### 1. 工作台

数字化工作台是企业 Web 端统一工作入口，提供：菜单入口、搜索、消息提醒、工作桌面、应用管理、导航设置、企业风格设置。

**关键能力**：
- **工作台设计**：所见即所得的在线布局设计，自由布局/流式布局，丰富的磁贴组件库
- **工作台主题**：业务经典主题、节日纪念日主题、企业定制主题、业务标准主题
- **企业风格设置**：全局风格（Logo/主题色/背景）→ 工作台主题 → 业务界面主题 → 业务偏好设置
- **人人看板**：个人工作台上关注业务指标，支持跨单据统一筛选（组织/项目/部门），预置指标卡和分析卡片
- **菜单管理**：功能地图（四叶草）+ 场景菜单（角色工作台），支持按应用/服务/菜单三级管理

**授权体系**：工作台按组织、角色、人员分配权限。支持个人工作台编辑（用户级设置 > 租户级配置）。

#### 2. 日志管理

三种日志：登录日志、操作日志、业务日志。支持 ES 智能搜索、统计报表和 Open API 接入。

**应用价值**：防止系统被盗用/破坏、运维可追溯、促进产品改进。

#### 3. 预警与调度

**预警**：在一定的数据条件下，定时执行某项业务操作，并将消息发送给对应人员。支持租户级/系统级预警类型，可配置条件阈值、定时规则、消息发送设置。

**调度**：业务系统中定时执行的操作（考勤日报生成、库存计算、成本结转等），通过 Quartz 调度框架，支持限流控制、消息通知、监控看板、任务日志追踪。

### 三、工具集

#### 1. 数据导入导出

**导入能力**：
- 5 种模板形式，支持自定义导入模板
- 三层校验：前端校验 → 解析校验（全局+行级）→ 业务校验（平台+领域 CHECK 规则）
- 百万数据智能拆批、异步进程管理
- 支持文本/附件/图片数据类型导入
- 支持特征/特征组导入、参照翻译设置

**导出能力**：
- 支持列表导出、详情导出、更新导入的导出
- 自定义导出模板（选择导出字段）
- 导出加密机制、异步导出管理

**模板管理**：租户级管理，支持按组织/交易类型/用户/角色分配不同模板，按组织隔离。

#### 2. 配置迁移

通过技术中台的部署能力，实现开发→测试→生产三环境之间数据的全量迁移。

**流程**：来源构建 → 数据确认 → 目标部署。包括数据包构建、传输包管理、数据对比预览、数据备份/恢复、部署执行。

**应用价值**：加速实施进度（一次配置多环境使用）、生产环境常态化可靠运维（关闭配置权限降低安全风险）、系统变更可追溯审计。

**新增特性**：支持选择传输包（下拉选择框）、未校验时允许查看传输包数据、迁移清单支持批量移除。

#### 3. 自助配置工作台

**核心价值**：整合领先实践、智能化赋能、客户价值为先、重塑服务模式。

**架构**：
- **管理端**（案例/场景运营团队）：维护案例库、场景步骤、数据包映射、推荐案例、黄金租户
- **运行端**（实施顾问）：案例库首页 → 案例详情 → 实施智库 → 数据安装 → 配置导引
- **黄金租户**（样板间）：预置场景配置包、报表包

**实施顾问使用流程**：搜索/匹配案例 → 学习案例详情和实施智库 → 一键安装预配置数据包 → 按 SOP 指引步骤微调环境配置。
