# 编码规则

> BIP 编码规则配置指南：编码元素拼接、序列号生成与自增规则（17K 字）。

## 概述

业务对象编号由编码元素按规则拼接（如常量+日期+序列号→`TT-20150615-0001`）。流水依据变化时序列号重新计数。
