# 数据权限开发

> BIP 数据权限开发完整指南：权限维度、权限范围、受控对象的配置与开发接入（27K 字）。

## 概述

数据权限在主组织权限隔离基础上，对业务数据进行更精细化的访问控制。与 MDD 开发模式结合，通过元数据字段标记声明数据权限过滤。

## 核心概念

| 概念 | 说明 |
|------|------|
| 权限管理维度 | 系统预置，描述数据权限受控列类型 |
| 权限范围 | 租户级配置，描述关联维度的数据取值范围 |
| 受控对象 | 需受控的应用/单据服务/档案（打标元数据） |

## 接入方式

通过 MDD 元数据字段增加标记，声明哪些业务字段需进行数据权限过滤。数据权限范围受限于主组织权限，不可逾越。
