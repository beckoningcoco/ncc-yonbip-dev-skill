# 单据开发入门

> 专业版新功能开发完整示例：以"商品采购需求单"为例，演示一主多子类型数据的对象建模、页面建模和交互规则配置。

## 适用场景

适用所有部署方案（公有云/私有云/专属云/本地部署），客户化定制 + ISV 生态开发。

## 业务需求（示例）

**列表页面**：付款方式多选、预估总金额和付款日期区间查询、需求单编码和采购组织固定列。

**单卡页面**：
- 卡片区分为三个分区：基本信息、采购信息、仓库信息
- 下部子页签：商品明细 + 附件

## 对象建模

### 1. 新建枚举

| 枚举名称 | 编码 | 内容 |
|----------|------|------|
| 前缀录入状态 | 前缀InputType | 1 手工录入、2 订单推送 |
| 前缀确认状态 | 前缀AffirmStatus | 1 待确认、2 确认中、3 已确认、4 已中止 |
| 前缀付款方式 | 前缀PayType | cash 现金、bank 转账 |

> 枚举保存即生效，无需发布。商开环境编码以 `_P` 结尾。

### 2. 创建业务对象

点击【新增业务对象】→ 录入名称与编码 → 勾选"同时生成主实体"。

**主实体设计**（引用接口：审批、自动编码、业务流、逻辑删除）：

| 字段编码 | 名称 | 类型 |
|----------|------|------|
| purOrg | 采购组织 | 单选引用：org.func.PurchaseOrg |
| inputType | 录入状态 | 单选枚举 |
| payType | 付款方式 | 多选枚举 |
| busiman | 业务员 | 单选引用：bd.staff.Staff |
| money | 预估总金额 | 数值 |
| payDate | 付款日期 | 日期 |
| attachments | 附件 | 附件 |

**子实体设计**（引用接口：业务流）：

| 字段编码 | 名称 | 类型 |
|----------|------|------|
| materialId | 物料 | 单选引用：pc.product.Product |
| materialCode | 物料编码 | 文本 |
| num | 数量 | 整数 |
| total | 预估金额 | 数值 |

### 3. 数据库表创建

实体发布后，在业务对象界面查看/导出 SQL 脚本，在引擎对应数据库中执行。如需其他数据库类型，通过导出按钮选择。

## 页面建模

### 新建页面

页面建模 → 选中业务对象 → 【新建页面】→ 选择"一主多子"单据类型 → 填入页面名称和编码（全局唯一）→ 勾选"生成列表页面" → 完成。

生成三类页面：列表页面、单卡页面、参照页面。

### 列表界面配置

- **查询区**：拖拽字段到常用查询条件，日期/金额类型比较符改为"区间"
- **列表区**：配置固定列（如采购组织、编码），拖拽调整字段顺序

### 卡片界面配置

**卡片分区**（右键卡片组 → 插入 → 卡片）：
- 基础信息：需求单编码、录入状态、来源单据号、业务员、币种、预估总金额、确认状态、付款方式、付款日期、备注
- 采购信息：采购组织、采购日期
- 仓库信息：库存组织

**多页签**：附件页签（表单 + 附件组件）、明细页签（子表 + 参照带出）。

### 关键配置

| 配置项 | 说明 |
|--------|------|
| 参照带出 | 物料参照选择后自动填充物料编码、计量单位到子表字段 |
| 字段精度 | 金额类型小数位默认 2 位 |
| 不可编辑 | 确认状态、预估总金额设为只读 |
| 占整行 | 备注字段设置占据整行显示 |

## 页面规则

> ISV 开发禁止使用页面规则与公式。

### 新增时赋默认值

页面状态勾选"新增"，触发时机"页面初始化"：
- 付款方式 → 常量 `转账`
- 采购日期 → 公式 `系统日期`
- 确认状态 → 常量 `待确认`

> 赋默认值场景下，配置迁移需先迁移默认值的档案，否则报错。

### 值改变求和

触发时机"值发生变化"，字段选择明细预估金额：
```
sum(fnProductPurchaseReqDetailList, "fnProductPurchaseReqDetailList.total")
```

## 常见问题

| 问题 | 解决 |
|------|------|
| 页面建模选不到实体元数据 | 确认实体已发布（含子实体） |
| 对象建模选不到引用实体 | 确认字段类型为单选/多选引用，被引用实体为可引用的单据/档案 |
| 找不到交互规则配置处 | 交互规则是页面级配置，选中左侧根部 → 右侧属性中找"页面规则" |
| 实体新增字段后页面没显示 | 手动在页面添加组件，关联实体字段（需刷新缓存） |
| 公式变量找不到字段 | 数字化建模 → 系统管理 → 业务对象 → 刷新元数据缓存 |
| 实体发布后 SQL 脚本为空 | 确认实体建模已填入数据库表名 |
| YDS 预览报 Nginx 502 | 确认引擎是本地运行的，编码符合开发规范 |
