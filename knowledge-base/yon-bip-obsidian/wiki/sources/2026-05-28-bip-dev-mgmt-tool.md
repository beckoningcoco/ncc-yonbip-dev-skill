## 开发管理工具用户手册

YonBIP V5.0.2507 开发管理工具用户手册（25 页），涵盖产品包管理和数据传输工具两大模块。

### 一、产品包管理

**核心价值**：开发成果物原环境制盘，多个目标环境无差异安装；迭代开发快速交付；按应用维度灵活迁移。

**操作流程**：
1. **创建发布项目**：新建项目（名称+编码）
2. **导入应用**：将应用构建创建的自建应用和原厂扩展应用导入发布项目
3. **系统数据管理**：查看和发布系统级数据
4. **发布应用**：发布后生成制品包，可上传到制品库创建产品盘
5. **下载制品**：将整个项目内容打包下载

**Git 集成**：支持将开发成果脚本提交到 Git 管理，需配置 Git 地址/分支/用户名/密码，也支持 SSH 协议（需上传私钥）

**迭代管理**：支持将脚本提交到指定迭代，管理迭代和查看提交日志

**产品盘生成**：直接生成产品盘，获取下载地址直接下载安装；生成可在后台运行，支持查看进度

### 二、数据传输工具

多环境协作的关键部分，实现环境间开发/验证/实施配置数据的迁移。

**传输包管理（源环境）**：
- **新建传输包**：填写名称和描述，自动生成唯一包编码（流转唯一依据）
- **添加数据**：选择数据类型→勾选数据→自动关联导出→包内查看
- **包状态**：无数据→未上载→已上载（已上载后不可再调整内容）

**已接入传输包的数据类型（24 种）**：
应用构建（扩展应用/元数据实体/枚举/查询元数据/单据页面/参照/函数脚本）、交易类型、工作流（流程模型）、业务流（业务流设计/单据转换规则）、模板（UI模板/参照模板/打印模板）、支撑服务（编码规则/编码映射/调度任务）、公共参数、工作台（自建应用/自建服务）、全球化（多语词条）、智多星 RPA（模板/机器人组/机器人/机器任务）

**传输包上载（源→目标）**：
- 根据包编码查找传输包，选择目标环境进行上载
- 可选择的目标环境范围为当前用户有权限的多环境下的租户
- 状态：上载中/成功/失败

**传输包部署（目标环境）**：
- **部署模式配置**：手动部署（默认）/ 自动部署（迁入时立即部署 / 每日定时部署）
- **手动部署流程**：待部署页签→确认来源包信息→确认导入方案→执行部署
- **导入方案**：业务流程设计支持"新增"和"覆盖"两种（可切换），其他数据类型按系统预制方式导入
- **状态跟踪**：待部署→部署中→成功/失败，可查看部署详情

### 环境协作流程

```
源环境                      目标环境
  传输包管理                  传输包部署
  (新建包→添加数据→打包)  →  传输包上载  →  (接收→校验→部署导入)
```
