# 开发体系

> YonBIP 社区文档中心 — BIP 专业开发体系总览。以架构图为主，文字为各模块标题与简介。

## YDS 核心开发模式

YDS 提供本地开发、发布和工程管理能力，覆盖端到端开发生命周期：

- **产品功能架构图**：YDS 整体模块组成
- **端到端开发生命周期**：从项目创建到部署上线的完整流程

## 远程调试（Remote Debug）

无需直接访问目标服务器，即可通过远程连接进行断点调试、日志分析与性能监控。

- 跨环境调试：本地 YDS 直接调试远程服务器应用
- 实时问题排查：不中断服务（多线程）定位生产问题
- 支持断点设置、单步执行等标准调试操作

## 本地调试

将线上服务复刻到个人电脑，在隔离的仿真环境中安全排查问题，避免线上问题无法本地复现。

## 制作产品盘及补丁

将服务代码、配置、脚本、依赖包、开发成果完整封装，实现标准化交付和一键部署。

## 同源架构

零代码、低代码、专业开发共用同一底层架构。

## API网关

完整 API 托管服务，覆盖 API 的创建、维护、发布、监控、安全管控等全生命周期管理。支持各种协议 API 的接入、统一管理和安全调用。

## 集成总线

新一代企业服务总线（ESB）：通过多种通信协议和连接器，连接并集成不同 SaaS 服务和企
业应用。基于云原生运行时引擎，提供可视化设计器和低代码脚本扩展的集成流程编排。
