# 特征开发

> BIP 特征（动态属性扩展）开发指南：通过特征机制为业务对象动态增加字段，无需修改实体模型（34K 字）。

## 概述

特征是 BIP 平台提供的动态属性扩展机制，允许在不修改原始实体元数据的情况下，为业务对象动态增加扩展字段，满足不同租户/客户的个性化需求。
