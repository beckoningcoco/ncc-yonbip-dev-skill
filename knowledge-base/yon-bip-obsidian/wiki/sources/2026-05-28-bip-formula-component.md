# 公式组件

> BIP 公式组件的使用指南：在 UI 模板和页面中配置公式进行数据计算（12K 字）。

## 概述

公式组件提供了可视化的公式配置能力，支持在 UI 模板中通过拖拽方式配置字段计算公式，实现金额合计、数量统计等场景。
