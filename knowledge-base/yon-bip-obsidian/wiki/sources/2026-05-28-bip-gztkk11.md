## 数字化工作台开发

对数字化工作台进行二次配置和开发，包括磁贴组件、个性化样式、修改平台 logo 和登录页背景等。仅适用于私有云和专属云的客户化定制/ISV 开发，公有云不支持。

## 核心概念

- **分析卡片**：磁贴的一种，非入口分析类，直接展示分析数据，受对应服务的数据和功能权限控制
- **应用**：可独立使用的最小产品集合（如假勤管理、总账等）
- **原子服务**：不可再分解的最小能力单元，也是权限管控的最小单元
- **磁贴**：常用功能入口贴到工作桌面，支持大小和样式自定义
- **菜单**：顶级菜单，对应后台各领域，最多 3 级菜单项

## 磁贴开发（React 组件模式）

工作台组件支持 YNF 组件和 React 组件两种模式，本文以 React 组件模式为例。

### 开发步骤

1. 下载脚手架：`iuap-apcom-widget-master.zip`
2. VS Code 打开前端工程，`npm install` 安装依赖
3. 终端执行 `npm run create <widget_name>` 创建磁贴组件（生成 `main.js` + `index.css`）
4. 在 `main.js` 中编写 React Class Component，使用 `axios` 请求后端数据
5. 在 `index.css` 中编写磁贴样式

### 代码示例

```javascript
class Widget extends Component {
  constructor(props) {
    super(props)
    this.state = { message: '员工类别', num: 0, text: '类' }
  }
  getPurchaseNum = () => {
    axios({
      method: 'post',
      url: 'https://bip-new15.yyuap.com/mdf-node/uniform/bill/querytree',
      params: { serviceCode: 'GZTBDM010', terminalType: 1, extend: true, locale: 'zh_CN' },
    })
  }
}
```
