## 跨租户工作通知

业务场景：供应商认证中，采购商需要给供应商发送工作通知（跨租户场景）。

## 两种发送场景

1. 采购商给采购商发送通知（同租户）— OpenAPI 可用
2. 采购商给供应商发送通知（跨租户）— OpenAPI 不支持，需用其他方式

## 同租户发送

通过 OpenAPI 发送工作通知：云平台 → 连接集成服务 → API 管理 → API 文档，搜索"工作通知"。后端代码集成需引入 `iuap-ip-openapi-sdk-8.2.100-RELEASE.jar`。

## 跨租户发送（OpenAPI 不支持）

OpenAPI 发送跨租户通知会导致供应商查看时自动切换到采购商租户，无法正确查看。需改用：

1. **Api 函数**：后端 Api 函数方式发送
2. **后端脚手架 SDK**（推荐）：方便代码统一管理
