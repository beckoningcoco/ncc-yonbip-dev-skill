## 事件中心开发介绍

事件中心是 BIP 中间件的核心组件，内核为 Kafka，主要目的是实现应用异步解耦、流量削峰、增强系统吞吐量。典型场景：单据异步导入导出、审批流事件监听、单据保存/审核后异步逻辑处理。

## 事件中心基本信息

- **地址**：`数字化工作台/iuap-apcom-event/event-center/index.html#/`
- **R2 地址变化**：换为 `数字化工作台/iuap-apcom-event/ucf-wh/eventCenter/index.html`

## EOS（应用可靠事件处理）对接

1. 在 YMS 控制台找到对应数据源，共享逻辑数据源
2. 勾选"事件中心消息发送服务"
3. 在 `iuap_yms_data.eventeos_task` 表插入映射关系（id 和 code 均为逻辑数据源名称）
4. 重启事件中心消息发送服务

## 数据库存储

所有表位于 `iuap_yms_data` 库：

| 表名 | 说明 |
|------|------|
| `new_event_type` | 事件类型主表 |
| `new_event_type_ext` | 事件类型扩展表（关联 event_type_id） |
| `new_event_listener` | 事件监听主表 |
| `new_event_listener_ext` | 事件监听扩展表（关联 event_listener_id） |
| `event_attributes_description` | 事件类型属性描述表（关联 event_type_id） |

### 查询示例

```sql
SELECT * FROM new_event_type WHERE event_type_name = 'APILINK签名审核';
SELECT * FROM iuap_yms_data.new_event_type_ext WHERE event_type_id = 7232223580191195136;
SELECT * FROM iuap_yms_data.new_event_listener WHERE listener_name = 'APILINK模板审核监听';
SELECT * FROM iuap_yms_data.event_attributes_description;
```
