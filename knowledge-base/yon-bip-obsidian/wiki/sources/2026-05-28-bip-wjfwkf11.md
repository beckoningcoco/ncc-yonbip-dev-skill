## 文件服务开发

BIP 文件服务的开发接入指南，涵盖文件上传、下载等基本操作。

## 关键内容

文件服务提供统一的文件存储和访问能力，支持通过 OpenAPI 和 SDK 两种方式接入。具体 API 可参考所在开发环境的 API 文档（云平台 → 连接集成平台 → API 网关）。
