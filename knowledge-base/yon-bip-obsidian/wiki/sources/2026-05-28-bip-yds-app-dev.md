# 新应用开发（YDS）

> 使用应用构建进行专业开发新功能的完整指南，涵盖对象建模、页面建模、编码规则、审批流、打印配置和应用发布。

## 适用场景

适用所有部署方案，客户化定制 + ISV 生态开发。

## 概念介绍

- **领域**：按业务序列划分的应用集合（如财务管理、采购供应等），自定义领域的主领域是"企业自定义"
- **YPD**：Yonyou Platform Development Framework，面向强类型实体（业务对象）以插件扩展开发模式为主的后端框架
- **导航层级**：领域 → 应用 → 分组 → 服务

## 快速新增实体字段格式

创建实体时可用逗号分隔格式批量添加字段：

**主实体**：
```
purOrg,purOrg,单选引用,org.func.PurchaseOrg;
inputType,inputType,单选;
payType,payType,多选;
busiman,busiman,单选引用,bd.staff.StaffNew;
currType,currType,单选引用,bd.currencytenant.CurrencyTenantVO;
money,money,数值;
payDate,payDate,日期;
attachments,attachments,附件;
remark,remark,文本;
```

**子实体**：
```
materialId,material,单选引用,pc.product.Product;
materialCode,materialCode,文本;
num,num,整数;
unit,unit,文本;
total,total,数值;
```

> 单选/多选字段需手动配置引用列。商开环境枚举以 `_P` 结尾，本地部署以 `_C` 结尾。

## 编码规则

### 概念

业务对象编号由时间、序列号、常量等按规则拼接。如 `TT-20150615-0001`：
- 常量 `TT-` + 时间 `20150615` + 常量 `-` + 序列号 `0001`
- **流水依据**：所有流水依据元素组合，序列号基于流水依据重新计数

### 配置

1. 数据模型勾选引用接口"自动编码"（自动生成编码字段）
2. 卡片页面选中编码字段 → 属性中【配置编码规则】
3. 新增编码元素：常量、属性值、系统时间、序列号
4. 保存后可在 数字化建模 → 编码规则定义 中查看

## 审批流

### 数据模型配置

引用接口勾选"审批" → 页面建模后审批功能自动生成（卡片+列表）。

### 流程定义

应用的"流程&自动化"界面：
1. 新建工作流 → 输入名称 → 确定
2. 点击【设计】→ 配置节点：参与人、抄送人、操作、时限、业务活动、智能审核、服务项
3. 【保存并发布】

### 应用效果

提交后生成待办消息 → 消息中心待办 → 点击审批 → 同意/退回。申请人可在详情页查看审批情况。

## 打印配置

1. 页面建模 → 打印模板 → 添加模板（可创建空白模板）
2. 设计界面：拖拽页眉、卡片组件 → 设置列数 → 选中数据字段 → 保存
3. 可配置多个模板，设置默认打印模板
4. 卡片页面自动生成打印按钮

## 公式

值改变求和公式：
```
sum(fn5ProductPurchaseReqDetailList, "fn5ProductPurchaseReqDetailList.total")
```

## 功能发布

开发完成 → 发布管理 → 添加分组 → 添加功能节点 → 发布应用。

> YDS 开发需制盘安装部署后菜单中才能被其他人访问。授权后等缓存刷新（约 10 分钟）。

## 常见问题

| 问题 | 解决 |
|------|------|
| 页面建模选不到实体元数据 | 确认实体已发布，主子类型需子实体也发布 |
| 对象建模选不到引用实体 | 确认字段类型正确 + 被引用实体为可引用的单据/档案 |
| 找不到交互规则配置处 | 选中左侧根部 → 右侧属性中找"页面规则" |
| 实体新增字段后页面没显示 | 手动添加组件关联字段，刷新缓存 |
| 公式变量找不到字段 | 数字化建模 → 系统管理 → 业务对象 → 刷新元数据缓存 |
