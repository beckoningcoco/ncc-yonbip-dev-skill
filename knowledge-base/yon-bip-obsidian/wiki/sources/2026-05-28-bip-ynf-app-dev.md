# 新应用开发（YNF）

> 基于 YNF（YonBuilder New Framework）+ YPD 全新开发框架的专业版新功能开发。**暂不支持 MDF/MDD 框架自建应用的升级**。

## 适用场景

适用所有部署方案。

## 应用构建

### 全新自建应用

**标准版**：空白构建 → 选择公共引擎
**专业版**：空白构建 → 选择自建引擎

创建方式：
- **从数据开始**：先建数据模型，再生成页面（推荐）
- **从页面开始**：智能生成表单 / 新建表单 / 新建仪表板（不推荐，会自动生成业务对象）

### 原厂应用扩展（YNF 新增）

在原厂标准化领域应用（财务、采购、人力资源等）基础上，选择对应原厂应用进行功能扩展。

## 对象建模（YNF 特色）

### 新建业务对象

新建方式选择"**实体模型**"，模型类型选择"**系统模型**"，支持**同时勾选主子实体**，选择子实体数量。

### 快速新增格式

```
字段编码,字段名称,类型,参照;
```

示例：
```
purOrg,采购组织,单选引用,org.func.PurchaseOrg;
money,预估总金额,数值;
```

### YNF 特色：右侧字段快捷栏

可直接在快捷栏设置字段、字段属性、接口信息等，无需进入编辑页面。

### 场景支持

在"场景"页签启用：审批流、业务流、打印、自动编码。

## 页面建模（YNF 差异）

### 新建表单

页面建模 → 【新建表单】→ **"从数据模型新建"** → 选择"一主多子" → 生成表单 → 自动生成列表页+卡片页。

### 查询区配置

YNF 使用 `FilterDateRangePicker` 组件，`compareLogic` 设为 `"between"`。

### 卡片分区

使用**"折叠面板"**组件（非 MDF 的卡片组），在折叠面板中插入"**表单**"组件承载字段。

### 页面赋值（YNF 替代页面规则）

**自定义赋值**支持三种配置方式：交互规则、公式、表达式。

- **新增赋值**：付款方式=`转账`、确认状态=`待确认`
- **求和公式**：直接在预估总金额字段上配置自定义赋值公式（不同于 MDF 的页面规则弹框）

## 功能发布（YNF 增强）

应用构建升级版 → 应用发布 → 选择单据 → 配置：

| 配置项 | 说明 |
|--------|------|
| 服务名称/编码 | 功能节点标识 |
| 图标 | 菜单图标 |
| 子页面 | 选择发布的子页面 |
| 权限配置 | **全员可见** / **自定义可见范围** / **角色授权** |

单据发布 → 下一步 → 选择导航目录和菜单分组方式 → 发布。

> 发布后页面修改无需再次发布（数据同源）。

## YNF vs MDF 核心差异

| 维度 | MDF | YNF |
|------|-----|-----|
| 应用构建 | 低代码开发-应用构建 | **应用构建升级版** |
| 新建业务对象 | 手动创建主子实体 | 可同时勾选主子实体+数量 |
| 卡片布局 | 卡片组 | **折叠面板 + 表单** |
| 查询区间组件 | 属性配置 | **FilterDateRangePicker** + compareLogic |
| 交互逻辑 | 页面规则 | **自定义赋值**（规则/公式/表达式） |
| 原厂扩展 | 无 | **原厂应用扩展** |
| 功能发布 | 分组+功能节点 | 选择单据+权限配置（全员可见/角色授权） |
| 框架后缀 | MDF + MDD | **YNF + YPD** |

## 常见问题

| 问题 | 解决 |
|------|------|
| 服务编码重复 | 添加后缀 |
| 角色编辑页找不到功能节点 | 客开单据属业务类角色，平台功能属管理类角色 |
| 页面建模修改后需再发布？ | 不需要（数据同源） |
| 选不到实体元数据 | 确认实体已发布（含子实体） |
| 公式变量找不到字段 | 数字化建模 → 系统管理 → 业务对象 → 刷新缓存 |
