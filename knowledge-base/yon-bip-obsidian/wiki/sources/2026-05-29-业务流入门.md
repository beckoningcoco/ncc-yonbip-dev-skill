# 业务流入门

> YonBuilder 业务流推单、拉单、回写的完整实操教程。

## 基本信息
- 来源：YonBIP 社区文档中心 (YWLRM1)
- 最后更新：2025-04-25

## 核心操作

### 推单（上游 → 下游）
1. 选中上游单据 → 拖拽连线到下游 → 点击"+"新建规则
2. 配置：基本信息 → 过滤设置（公式条件）→ 触发设置（手工/自动）→ 生单方式（新增单据/转单画面）→ 转换规则（字段映射）→ 消息设置
3. 操作：上游列表选中数据 → 下推 → 选规则按钮 → 填写下游 → 保存

### 拉单（下游 ← 上游）
1. 下游单据创建"拉单"列表页（含取消按钮+生单自动规则）
2. 在推单规则右侧"拉单"下新建规则 → 选择拉单画面 → 设置转换规则
3. 操作：下游列表点击拉单 → 选数据 → 生单 → 保存

### 回写（下游 → 上游）
1. 下游拖拽连线到上游 → 新建回写规则
2. 触发设置选单据动作（如"保存"）→ 配置回写字段
3. 效果：下游保存后自动更新上游对应字段

## FAQ
| 问题 | 答案 |
|------|------|
| 下游保存后不自动返回？ | 设计器 → 最上层 → 开启"保存停留当前页" |
| 下推提示"无可用操作"？ | 业务流需发布 + 页面编码=业务对象编码 |
| 附件支持推拉单？ | 支持 |
| 跨单据回写（1→2→3，3回写1）？ | 不支持，3只能回写2 |
