# 创建应用

> YonBuilder 自建应用的创建和管理：新建、编辑、停用、删除、发布、权限控制。

## 基本信息
- 来源：YonBIP 社区文档中心 (YYGJSY11)
- 最后更新：2026-01-06
- 入口：云平台 > 开发平台 > 低代码开发 > 应用构建

## 核心操作

### 新建应用
新建应用 → 选择空白模板（或扩展原厂节点）→ 输入名称/领域/引擎/图标 → 完成

### 应用管理
- 编辑：修改名称、领域
- 删除：应用管理界面选择删除
- 权限：成员管理 → 仅成员可见 → 添加成员

### 应用设计阶段
进入应用卡片后五大模块：数据建模 → 页面建模 → 流程&自动化 → 集成配置 → 发布管理

## 应用构建 vs UI模板
应用构建可额外支持：脚本、按钮、流程自动化等。UI模板仅能调整表单布局。

## FAQ
| 问题 | 答案 |
|------|------|
| 限制可见范围？ | 成员管理设仅成员可见 |
| 删除应用？ | 应用管理界面删除 |
| 扩展是什么？ | 扩展原厂功能（脚本、按钮等） |
| 修改图标？ | 编辑界面修改 |
