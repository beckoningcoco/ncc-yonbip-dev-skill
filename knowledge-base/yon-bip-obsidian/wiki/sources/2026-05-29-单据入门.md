# 单据入门

> 手把手教学：在 YonBuilder 中创建应用、业务对象、单卡/主子表/树形表页面，以及发布授权全流程。

## 基本信息
- 来源：YonBIP 社区文档中心 (DJRM1)
- 最后更新：2025-10-16

## 核心操作流程

### 1. 新建应用
搜索"应用构建"→ 新建应用 → 输入名称/选择领域 → 完成

### 2. 单卡页面
对象建模：新建业务对象 → 编辑实体（新增字段）→ 保存并发布
页面建模：选中业务对象 → 新建页面 → 单卡 → 选元数据 → 预览

### 3. 主子表页面
对象建模：新建业务对象 → 创建主实体 + 子实体 → 发布
页面建模：一主多子 → 选元数据 → 完成

### 4. 树形表页面
关键配置：实体引用接口勾选"自动编码"和"树形结构"，取消"审批"和"业务流"

### 5. 应用发布与授权
发布管理 → 增加分组 → 选择页面 → 发布应用 → 角色管理授权菜单

## 常见问题速查
| 问题 | 答案 |
|------|------|
| 字段类型发布后可改？ | 不可，需删字段重建 |
| 长文本用什么类型？ | 大文本 |
| 如何唯一性校验？ | 实体编辑 → 唯一性校验 → 新增规则 |
| 子表自动增行？ | 页面设计器 → 子表 → 自动增行开 |
| 子表非必填？ | 设计器 → 子表 → 勾选"子表为空" |
