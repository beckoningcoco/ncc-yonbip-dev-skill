# 审批流入门

> YonBuilder 审批流完整教程：从单据构建到流程配置到测试验证。

## 基本信息
- 来源：YonBIP 社区文档中心 (SPLRM1)
- 最后更新：2025-10-16

## 操作步骤

### 1. 单据构建
对象建模：创建业务对象/实体，实体引用接口勾选"审批" + 场景支持勾选"审批流"
页面建模：创建一主多子页面

### 2. 流程配置
流程&自动化 → 新建流程 → 设计进入审批流设计器
- 拖拽"普通环节"到画布
- 双击分支线设置流程走向逻辑（条件分支）
- 配置每个环节的审批人
- 保存并发布

### 3. 测试验证
维护业务数据 → 提交 → 卡片页面点击审批

## FAQ
| 问题 | 答案 |
|------|------|
| 提交后审批按钮没反应？ | 单据编码必须和业务对象编码一致 |
| 支持无流程审批？ | 支持，列表加批量审核/弃审按钮，详情加审核/弃审按钮 |
| 按交易类型走不同审批流？ | 支持，可按组织维度/交易类型维度分别创建流程 |
| 控制审批人对字段的编辑权限？ | 流程设计器右上角"流程字段权限" |
| 退回提交方式？ | 开始节点可配置：直接返回本节点 / 重新依次审批 / 退回环节选择 |
