# 对象建模

> YonBuilder 对象建模完整操作指南：业务对象、实体、场景支持、枚举管理、特征管理。

## 基本信息
- 来源：YonBIP 社区文档中心 (DXJM12)
- 最后更新：2026-01-07

## 业务对象与实体

### 创建规则
- 一个业务对象只能创建一个主实体
- 编码：字母开头，字母/数字/下划线，≤60位（用业务流则≤13位）
- 默认勾选"同时生成主实体"

### 实体引用接口（重要：必须在创建页面前勾选！）
| 接口 | 作用 | 自动增加 |
|------|------|---------|
| 审批 | 接入工作流 | 提交/撤回/审批流按钮 |
| 业务流 | 接入业务流 | 下推按钮 |
| 交易类型 | 启用交易类型 | 交易类型字段 |
| 自动编码 | 接入编码规则 | 编码字段 |
| 档案状态 | 状态管理 | 启用/停用 |
| 主组织 | 主组织权限 | 主组织参照字段 |
| 树形结构 | 构建树形表 | 父级/路径/层级字段 |

### 字段关键字（不可用）
approvalFlow、base、bizFlowName、code、dr、id、pubts、status、tenant_id、version 等 40+ 个系统保留字

### 常用参照 URI
| 名称 | URI |
|------|-----|
| 组织 | org.func.BaseOrg |
| 部门 | bd.adminOrg.AdminOrgVO |
| 员工 | bd.staff.Staff / StaffNew |
| 物料 | pc.product.Product |
| 供应商 | aa.vendor.Vendor |
| 客户 | aa.merchant.Merchant |

### 文本存储
- 文本类型：varchar(200)
- 大文本：longText

## 场景支持（11项）
打印、编码规则、审批流、业务流、报表、消息模板、交易类型、自定义项、导入导出、预警任务、规则引擎
——每项需单独勾选才能在对应菜单中使用

## 特征管理（三种）
1. **自定义项特征组**：弹性扩展实体属性 → 创建特征实体→实体加字段→场景勾选→创建特征→分配→启用
2. **物料自由项特征组**：与物料关联，需物料引用字段
3. **物料选配特征组**：与物料关联，只支持数值类型特征

存储：每个特征实体在数据库对应三张宽表（_1, _2, _3）

## FAQ 关键点
- 实体引用接口勾选必须在创建页面前完成，否则需重建页面
- 唯一性校验仅比较数据库已有数据，子表需额外前端脚本校验
- 字段类型发布后不可修改
