# 应用构建升级版快速入门

> 应用构建升级版从 0 到 1 教程：智能生成表单、数据驱动建模、业务流/工作流配置。

## 基本信息
- 来源：YonBIP 社区文档中心 (KSRM-YYGJSJB1)
- 最后更新：2025-10-16
- 入口：云平台 > 开发平台 > 低代码开发 > 应用构建升级版

## 两种建模方式

### 从页面开始（智能生成表单）
格式要求填充左侧 → 点击"智能识别" → 右侧自动生成字段/组件 → 插入完成页面创建

### 从数据开始
对象建模 → 新建业务对象 → 编辑实体新增字段 → 保存并发布 → 页面建模 → 新建表单 → 生成表单

## 流程管理
- **业务流**：多业务间流转，下推/拉单规则配置（触发设置、生单方式、转换规则自动匹配）
- **工作流**：审批流程设置，支持选人（直接选人/按业务数据/按发起人）

## 核心概念
- **应用构建升级版**：新版本，一个应用可同时含零代码/低代码/专业开发三种模式，支持多角色协同，零代码可平滑升级到低代码/专业开发
- **系统模型**：主实体、主子实体、主子孙实体、树实体
- **业务模型**：预置的业务单据模型（如财务过账）
