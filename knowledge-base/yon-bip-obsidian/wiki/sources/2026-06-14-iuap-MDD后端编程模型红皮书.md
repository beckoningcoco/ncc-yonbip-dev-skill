# iuap MDD 后端编程模型红皮书

> 133 页，90K 字 | 6 个核心部分

## 核心内容

用友 iuap MDD (Model-Driven Development) 后端编程模型，是 BIP 旗舰版后端开发的基石。

## 六大核心部分

1. **数据模型**：实体定义、VO 设计、ORM 映射、数据库操作
2. **业务逻辑**：Service 层设计、事务管理、业务规则
3. **API 服务**：RESTful 接口设计、Controller 编写、参数校验
4. **事件处理**：事件发布/订阅机制
5. **安全与权限**：认证授权、数据权限
6. **扩展机制**：插件体系、扩展点

## 关键概念

- **MDD**：Model-Driven Development，模型驱动开发
- **DAO 模式**：数据访问层设计
- **Service 模式**：业务逻辑层设计
- **VO/DO 转换**：Value Object 与 Data Object 的映射
