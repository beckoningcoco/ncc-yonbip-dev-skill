# iuap MDF 前端编程模型红皮书

> 1846 页，1.1M 字 | BIP 前端开发核心参考

## 核心内容

用友 iuap MDF (Model-Driven Frontend) 前端编程模型，是 BIP 旗舰版前端开发的完整参考手册。

## 核心架构

- **ViewModel**：前端数据模型，和元数据实体绑定
- **数据绑定**：ViewModel ↔ UI 控件双向绑定
- **事件系统**：页面生命周期、控件事件
- **命令系统**：按钮命令、单据命令

## 关键 API

- `viewModel.on()` — 事件监听
- `viewModel.get()` — 获取字段 ViewModel
- `viewModel.execute()` — 执行命令
- `cb.utils` — 平台工具函数
- `cb.rest` — 平台 REST 调用
