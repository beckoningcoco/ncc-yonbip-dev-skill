>  请求方式	POST | Fixed Asset Consolidation (EFI)


## 1. 请求说明

请求地址	/yonbip/EFI/FixedAssetMerge/delete
请求方式	POST
ContentType	application/json
应用场景	开放API
事务和幂等性	无
用户身份	支持传递普通用户身份，详细说明见开放平台用户认证接入规范
来源	系统级

名称	类型	参数位置	必填	描述

Body参数
名称	类型	数组	必填	描述

id	string	否	否	单据id,单据id和单据编码不能同时为空,id和编码同时存在,以id为准 示例：1796677711656648705
code	string	否	否	单据编码,单据id和单据编码不能同时为空,id和编码同时存在,以id为准 示例：HB00048
