# 检测室删除

>  请求方式	POST | Testing Room (IMCS)


## 1. 请求说明

请求域名	动态域名，获取方式详见 获取租户所在数据中心域名
请求地址	/yonbip/imcs/inspectioncenter/delete
请求方式	POST
ContentType	application/json
应用场景	开放API
事务和幂等性	
用户身份	支持传递普通用户身份，详细说明见开放平台用户认证接入规范
来源	系统级

名称	类型	参数位置	必填	描述
access_token	string	query	是	调用方应用token
Body参数
名称	类型	数组	必填	描述
data	object	是	是	检测室
id	long	否	否	检测室id,id和code必填其中一项，同时填入以id为准 示例：1623598896206315520
code	string	否	否	检测室编码,id和code必填其中一项，同时填入以id为准 示例：0002
orgId	string	否	否	质检组织id，传值为id或code皆可，单组织不填,多组织必填 示例：LN-002
