## 1. 请求说明

请求地址	/yonbip/uspace/rest/v1/identity/publicMoney/verify
请求方式	POST
ContentType	application/json
应用场景	开放API
事务和幂等性	无
来源	系统级

## 2. 请求参数

名称	类型	参数位置	必填	描述
access_token	string	query	是	调用方应用token
Body参数
名称	类型	数组	必填	描述
flowId	string	否	是	实名认证流程ID，来源自打款接口返回值的verifyId字段 示例：73577c969f9e4c488ebeec676c4e7
amount	string	否	是	金额 示例：0.01
companyAppAuthId	string	否	是	待认证的企业授权ID 示例：18062377246271733

## 3. 请求示例

Url: /yonbip/uspace/rest/v1/identity/publicMoney/verify?access_token=访问令牌
Body: {
	"flowId": "73577c969f9e4c488ebeec676c4e7",
	"amount": "0.01",
	"companyAppAuthId": "18062377246271733"
}
