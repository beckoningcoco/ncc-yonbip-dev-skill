# 离职获取模板配置

>  请求方式	POST | Employee Management - Self-Service (CHR)


## 1. 请求说明

请求域名	动态域名，获取方式详见 获取租户所在数据中心域名
请求地址	/yonbip/HCC/yonbip-hr-staffmgr/self_dimission/getTpl
请求方式	POST
ContentType	application/json
应用场景	开放API
事务和幂等性	无
用户身份	支持传递普通用户身份，详细说明见开放平台用户认证接入规范
来源	系统级

名称	类型	参数位置	必填	描述
access_token	string	query	是	调用方应用token
isQueryUnion	boolean	query	否	查询移动端模板需要传改参数值为true    默认值: false
Body参数
名称	类型	数组	必填	描述
billno	string	否	否	单据编码 示例：self_dimissionbillcard 默认值：self_dimissionbillcard
terminalType	string	否	否	终端类型 1 PC 3 移动 示例：1
transtype	string	否	否	交易类型id 示例：1921775059325682949
uiAssignDataMap	object	否	否	uiAssignDataMap
orgId	string	否	否	orgId 示例：1921226884064477190
serviceCode	string	否	否	serviceCode 示例：hresh_dimission_list
tplmode	number
小数位数:0,最大长度:10	否	否	tplmode 2 PC 4 移动 示例：2
