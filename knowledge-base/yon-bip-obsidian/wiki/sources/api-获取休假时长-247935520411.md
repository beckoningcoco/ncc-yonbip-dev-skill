# 获取休假时长

>  请求方式	POST | Leave Application Form (TIME)


## 1. 请求说明

请求域名	动态域名，获取方式详见 获取租户所在数据中心域名
请求地址	/yonbip/HCC/time/leave/queryLeaveLength
请求方式	POST
ContentType	application/json
应用场景	开放API
事务和幂等性	MDD幂等
用户身份	支持传递普通用户身份，详细说明见开放平台用户认证接入规范
来源	系统级

## 2. 请求参数

名称	类型	参数位置	必填	描述
access_token	string	query	是	调用方应用token
Body参数
名称	类型	数组	必填	描述
beginDate	string	否	是	开始时间 示例：2026-02-04 06:00:00
endDate	string	否	是	结束时间 示例：2026-02-07 00:01:00
leaveTypeId	string	否	是	请假类型ID 示例：0000L6YTYEY5FUZPXE00000000000002
userId	string	否	否	用户userId（staffId、staffCode、userId三选一必填，有staffId时优先使用staffId，没有staffId时优先使用staffCode，如果staffId和staffCode都没有，则使用userId） 示例：1952441692752707593
staffId	string	否	否	员工id（staffId、staffCode、userId三选一必填，有staffId时优先使用staffId，没有staffId时优先使用staffCode，如果staffId和staffCode都没有，则使用userId） 示例：1952441692752707593
staffCode	string	否	否	员工code（staffId、staffCode、userId三选一必填，有staffId时优先使用staffId，没有staffId时优先使用staffCode，如果staffId和staffCode都没有，则使用userId） 示例：001
