# 参照查询方案和模糊查询切换

> 来源：https://c2.yonyoucloud.com/iuap-yonbuilder-designer/ucf-wh/docs-mdf/mdf/index.html#/guides/05-canzhaochaxunfanganhemohuchaxunqiehuan | 所属：指南

# [](#参照查询方案和模糊查询切换)参照查询方案和模糊查询切换

参照pub_ref表的extendField字段配置 showSwitchFilterIcon为true（布尔类型）（默认显示查询方案）

如果需要默认显示模糊搜索 则需要在参照pub_ref表的extendField字段配置 defaultShowFilter为false （布尔类型）
