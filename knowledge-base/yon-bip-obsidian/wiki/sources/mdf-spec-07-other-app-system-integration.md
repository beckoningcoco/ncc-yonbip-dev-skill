# [](#与其它壳应用集成)与其它壳应用集成

## [](#1-离线包集成)1. 离线包集成

可通过下载离线资源包的方式将MDF资源下载到本地，通过本地加载打开MDF单据，此方式免去资源下载的过程，打开页面速度有明显提升

## [](#2-轻应用集成)2. 轻应用集成

通过MDF单据链接，远程加载MDF资源并打开单据页面，此方式每次打开页面都会重新加载一遍资源，虽然有浏览器缓存，但仍然比较慢
