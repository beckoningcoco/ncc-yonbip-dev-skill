# [](#多语文本框-inputmultilang)多语文本框 InputMultiLang

用于在一个文本框中录入同一段信息的多种语言翻译。用户可以在不同的语种输入相应的文本，以便系统能够在多语言环境中显示适当的翻译内容。

import { InputMultiLang } from "ynf-lib-base"

## [](#何时使用)何时使用

- 多语言应用程序: 适用于需要支持多种语言的应用程序，例如国际化的网站或应用;
- 翻译系统: 在翻译管理系统中，录入同一段文本的多种语言版本，以便后续翻译和发布。
