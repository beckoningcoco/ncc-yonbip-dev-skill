# [](#链接-link)链接 Link

允许用户通过点击链接快速跳转到其他页面、网站等。

import { Link } from "ynf-lib-base"

## [](#何时使用)何时使用

- 用于页面与页面之间的交互、嵌入内部或外部链接。
