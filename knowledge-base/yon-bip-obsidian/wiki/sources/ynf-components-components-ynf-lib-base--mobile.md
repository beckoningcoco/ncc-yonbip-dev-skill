# [](#电话-mobile)电话 Mobile

用于在前端表单中录入电话号码。该组件支持各种国际格式的电话号录入，并具有实时校验功能，以确保用户输入的电话号码符合有效的格式。

import { Mobile } from "ynf-lib-base"

## [](#何时使用)何时使用

- 注册和登录表单：在用户注册或登录时录入手机号码，并确保格式的准确性;
- 全球应用：支持国际化的应用场景，如跨国业务、旅行网站等，需要处理来自不同国家的电话号码。
