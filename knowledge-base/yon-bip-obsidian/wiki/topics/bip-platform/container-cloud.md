# 容器云（YKS）

DNS API：Token 获取 → 域查询 / 插入 / 删除（RESTful）。

多云接入：通用 K8s（kubeconfig）、阿里云 EDAS、腾讯云 TSF。云中立代理：Token 认证。

镜像异步推送 API。YMS 配置：`yms_config_download` + serviceCode。
