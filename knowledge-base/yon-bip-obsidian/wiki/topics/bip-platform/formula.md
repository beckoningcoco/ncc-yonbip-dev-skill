# 公式组件

设计态公式编辑器 → AST → 运行态公式引擎 → 计算结果。

三大组成：函数类型（内置 + 自定义）、变量类型（单据字段 / 系统变量 / 上下文变量）、公式编辑器。

触发方式：页面事件（字段值变更）、MDD 规则嵌入、API 调用。

自定义扩展：`@FormulaFunction` 注册函数、`@FormulaVariable` 注册变量。支持多语言。
