# 开放平台 API 网关

## 网关能力

| 能力 | 说明 |
|------|------|
| 限流 | 按 API/租户/应用维度 |
| 熔断 | 失败考察阈值 + 考察期 + 半开错误阈值 + 熔断时长 |
| IP 黑白名单 | 访问控制 |
| 插件扩展 | 热部署，无需重启 |
| API 日志 | ES 存储，保留 2 个月 |
| 网关监控 | API 级 + 网关级双维度 |

## 四种插件类型

| 插件 | 用途 |
|------|------|
| 安全认证插件 | 自定义鉴权 |
| 业务扩展插件 | 参数转换、业务校验 |
| 返回值插件 | 自定义响应格式 |
| 访问控制插件 | IP/时间/频率限制 |

支持热部署，无需重启服务。

## API 监控指标

| 指标 | 说明 |
|------|------|
| 调用总次数 | 可下钻查看 API 调用详情 |
| 调用成功/异常次数 | 区分 4xx（客户端）/ 5xx（服务端） |
| API 响应时间 | 平均值/最大值/最小值 + 趋势折线图 |
| 网关耗时 / 业务耗时 | 区分网关层和业务层延迟 |
| 流入/流出流量 | 合计/平均/最大/最小 |
| API 调用租户数 | 可下钻查看租户列表 |
| API 调用应用数 | 可下钻查看 AppKey 列表 |

## 熔断策略

可配置：失败考察阈值、失败考察期、半开错误考察阈值、半开考察期、熔断时长。

## API 日志

ES 存储，记录 TraceId、APIID、请求 URL/IP、调用时间、租户、请求/响应字节数、响应状态、请求参数、后端请求参数、返回值。保留最近 2 个月。
