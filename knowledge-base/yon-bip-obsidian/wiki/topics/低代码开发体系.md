# 低代码开发体系

> 涵盖 YonBIP 低代码开发的完整能力体系：概念、架构、模块、工具、流程、学习路径。

## 体系架构

低代码开发分为两大分支：

1. **[[可视化应用构建]]** → 新建 [[自建单据]]
2. **[[可视化应用扩展]]** → 扩展 [[原厂单据]]（即 [[原厂单据扩展开发]]）

## 覆盖的能力模块

### 数据层
- [[业务对象]]、[[实体]]、[[枚举]]
- [[特征]]（动态属性扩展）
- [[元数据]]（描述实体结构）
- [[档案]]（基础核心数据）

### 页面层
- 单据开发（页面设计器、公式、页面规则）
- 参照开发（自定义参照、过滤、联想匹配）
- 报表开发（智能分析、语义模型）
- 移动端（单卡/一主多子/列表）

### 逻辑层
- [[前端脚本]]（MDF/YNF 预置函数、生命周期、模态框）
- [[后端脚本]]（YonQL、远程调用、消息、定时任务）
- 规则链（上下文信息处理）

### 流程层
- [[工作流]]（审批流转）
- [[业务流]]（推单/拉单/回写）

### 权限层
- [[功能权限]]（谁能用哪些功能）
- [[数据权限]]（能看到/操作哪些数据）

### 运维层
- 配置迁移（开发成果迁移至其他租户）
- 应用发布（部署到工作台）

## 学习路径

1. 理解核心概念 → 阅读 [[低代码开发能力介绍]]
2. 学习对象建模 → [[实体]]、[[业务对象]]、[[元数据]]
3. 掌握页面开发 → [[前端脚本]]、页面设计器
4. 深入后端逻辑 → [[后端脚本]]、[[YonQL]]
5. 配置流程 → [[工作流]]、[[业务流]]
6. 实践项目 → [案例汇总](https://community.yonyou.com/article/detail/5270)
