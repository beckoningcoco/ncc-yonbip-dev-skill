# 问题处理

> BIP 开发中常见报错和解决方案汇总。

## 启动类问题

| 报错 | 原因 | 解决 |
|------|------|------|
| [[2026-05-27-yms-start-error\|请求YMS控制台失败]] | 配置文件路径错误/未连VPN/HTTPS问题 | 检查配置、连VPN、换HTTP |
| [[2026-05-27-datasource-error\|找不到数据源]] | yms_middleware.json 中 IP 不正确 | 找运维确认正确的 IP |
| [[2026-05-27-servlet-error\|ServletWebServerApplicationContext]] | 缺少 spring-boot-starter-web 依赖 | pom.xml 添加依赖 |

## API 调用类

| 报错 | 原因 | 解决 |
|------|------|------|
| [[2026-05-27-api-user-bind\|开放平台接口需要绑定用户]] | 机器人身份 token 无权限 | 手动覆盖 token 或新建线程 |
| [[2026-05-27-orgid-permission-error\|OrgId 无使用权限]] | 同上，身份绑定问题 | 参考开放平台用户绑定方案 |
| [[2026-05-27-api-issues\|tradeRouteID 无法保存]] | isEndTrade 字段为空 | 同时传 tradeRouteID + isEndTrade |

## 影像上传类

| 报错 | 原因 | 解决 |
|------|------|------|
| [[2026-05-27-img-upload-error\|非专岗环节不允许上传]] | 审批流未进入专岗 | 循环等待 ispostnode=true |
| [[2026-05-27-img-task-error\|获取不到单据类型标识]] | appKey/appSecret 用错环境 | 检查环境对应配置 |

| 报错 | 解决 |
|------|------|
| [[2026-05-27-button-not-show\|新增按钮不显示]] | 角色权限 + 同步按钮 + 清缓存 |
| [[2026-05-27-billcontext-error\|BillContext 为空]] | 检查调用地址参数 |
