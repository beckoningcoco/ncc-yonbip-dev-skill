# YDS出盘制盘

## 1. Git 拉取最新代码

## 2. YDS 下载脚本

先到**产品包管理**节点，对每个模块点"发布"（确保使用最新脚本）。

![image-20260415101932674](YDS出盘制盘/image-20260415101932674.png)

![image-20260415101959977](YDS出盘制盘/image-20260415101959977.png)

然后执行下载脚本命令，成功后工程会多出脚本目录结构。

![image-20260116180238144](YDS出盘制盘/image-20260116180238144.png)

![image-20260116180303950](YDS出盘制盘/image-20260116180303950.png)

## 3. 制盘

1. **删除旧模板**：如果制盘视图有旧模板，先删除，重新生成模板
2. **勾选出盘项目**：出脚本需勾选两个（项目+脚本），不出脚本只勾选项目
3. **填写必填信息**：右边必填字段一般会自动填充默认值，如未填充则关闭页签重新点击

![image-20260116180535862](YDS出盘制盘/image-20260116180535862.png)

![image-20260116180633352](YDS出盘制盘/image-20260116180633352.png)

![image-20260415102155338](YDS出盘制盘/image-20260415102155338.png)

![image-20260415102243627](YDS出盘制盘/image-20260415102243627.png)

4. 点击"制盘"，生成的压缩包即为最终补丁盘。

![image-20260415102429070](YDS出盘制盘/image-20260415102429070.png)

![image-20260415102500665](YDS出盘制盘/image-20260415102500665.png)
