# YDS安装用友的AI插件

## 1. 下载 YDS 开发工具

- 下载 **JDK21** 版本的 YDS 开发工具（AI 插件最低要求 JDK17）
- 版本要求：2026年3月以后的版本（之前可能存在 BUG）
- 下载地址：[YDS下载中心 - YonBIP开发者社区](https://community.yonyou.com/yds/download)

## 2. 安装插件

打开 YDS 工具，安装两个插件：
- YonBoss AI 插件
- 对应的辅助插件

## 3. 配置插件

### 第一步：基础配置
在 YDS 工具的设置中配置插件的基础连接信息。

### 第二步：环境配置
配置 YDS 当前的开发环境信息。

### 第三步：导入 JSON 并配置 API
- 下载 JSON 文件
- 到 **API发布** 节点，选择菜单 → 导入 JSON 文件 → 自动导入 6 个 API
- 到 **API调用** 节点，找到 YDS 工具中配置的 appKey，授权刚添加的 6 个 API

## 4. 开始使用

1. 启动 YDS 服务
2. 点击侧边栏小机器人图标，登录
3. 登录后即可开始使用

> 如果习惯在右边使用，可以将面板拖动到右侧。
