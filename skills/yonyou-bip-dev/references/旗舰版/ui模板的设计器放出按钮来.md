# ui模板的设计器放出按钮来

在 UI 模板设计器中，把页面地址里的 `sceneData=simple` 替换成 `sceneData=domain`，复制链接新页面打开即可看到按钮等更多组件。

```
https://gxdev.lzlj.com/iuap-yonbuilder-designer/ucf-wh/designer/apps/quickdev/index.html#/onepage/design/EAR/PT2401405945907773444/ysmdd?sceneData=domain&busiObj=collection&serviceCode=ear_bill_collection&feDomainKey=yonbip-fi-earapbill&orgId=666666&locale=zh_CN
```
