# 修改事项分录传送状态字段

## 调用 API

```java
// 调用接口，将传送状态改为传送失败
Map<String, Object> reqMap = new HashMap<>();
reqMap.put("code", "209"); // 200 是成功
reqMap.put("msg", "");
Map<String, Object> dataMap = new HashMap<>();
Map<String, Object> dataInfoMap = new HashMap<>();
dataInfoMap.put("busiId", infoMap.get("busiId"));
dataInfoMap.put("billVersion", infoMap.get("busiVersion")); // 版本号不能乱填
dataMap.put("busiId", infoMap.get("busiId"));
dataMap.put("id", UUID.randomUUID().toString().replace("-", ""));
dataMap.put("code", UUID.randomUUID().toString().replace("-", ""));
reqMap.put("dataInfo", dataInfoMap);
reqMap.put("data", dataMap);
Map map = yonBipUtil.sendEventVoucher(reqMap);
```

## 数据库

涉及两张表：

```sql
-- 生成总账凭证节点，修改 target_send_status 字段
SELECT * FROM fieaai.aai_data_clue T0
WHERE T0.ytenant_id = 'vc2c07rl'
  AND T0.voucher_id IN ('2424245027211313160','2424270135590125577')

-- 事项分录节点，"send_gl_state" 0 是成功 4 是传送中
SELECT * FROM fieaai.aai_voucher
WHERE voucher_no = 'S25121100001'
```
