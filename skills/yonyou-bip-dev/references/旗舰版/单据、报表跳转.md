# 单据、报表跳转

## 场景

1. 跳转自建单据（pc、mobile）
2. 跳转原厂单据（pc、mobile）
3. 单据跳转报表
4. 报表跳转单据

## 方式

1. `window.open`
2. `cb.loader.runCommandLine`
3. 工作台打开：`window.jDiwork.openService`

## 关键参数

- **业务中台域名**：租户所在数据中心域名，例如 dbox.yonyoucloud.com
- **单据类型 billType**：Voucher / VoucherList
- **单据编码 billNo**：UI 模板中获取
- **服务域 domainKey**：自建单据为 developplatform，原厂单据按实际获取
- **服务编码 serviceCode**：单据发布后才有

## 一、window.open（PC 和 Mobile）

### URL 模板

```
// 列表页
https://<域名>/mdf-node/meta/<单据类型>/<单据编码>?domainKey=<服务域>&serviceCode=<服务编码>

// 详情页新增态
https://<域名>/mdf-node/meta/<单据类型>/<单据编码>/add?domainKey=<服务域>&serviceCode=<服务编码>

// 详情页浏览态
https://<域名>/mdf-node/meta/<单据类型>/<单据编码>/<主键>?domainKey=<服务域>&serviceCode=<服务编码>

// 详情页编辑态（先打开浏览态，再用脚本触发编辑）
viewModel.biz.do('edit', viewModel);
```

### 跳转自建单据

```js
// 跳转列表
window.open("https://dbox.yonyoucloud.com/mdf-node/meta/VoucherList/pageRouteList?domainKey=developplatform&serviceCode=2110474997617131529")

// 跳转卡片新增态
window.open("https://dbox.yonyoucloud.com/mdf-node/meta/Voucher/pageRoute/add?domainKey=developplatform&serviceCode=2110474997617131529")

// 跳转卡片浏览态（2099323913899606020：目标单据id）
window.open("https://dbox.yonyoucloud.com/mdf-node/meta/Voucher/pageRoute/2099323913899606020?domainKey=developplatform&serviceCode=2110474997617131529")
```

Mobile 跳转在 URL 末尾加 `&isMobile=true`，路径中 VoucherList→Voucherlist, Voucher→Voucher, pageRoute→pageRouteMobileList/pageRouteMobileArchive。

### 跳转原厂单据（以销售订单为例）

```js
// 列表
window.open("https://dbox.yonyoucloud.com/mdf-node/meta/voucherlist/voucher_orderlist?domainKey=udinghuo&serviceCode=orderList")

// 卡片新增态
window.open("https://dbox.yonyoucloud.com/mdf-node/meta/voucher/voucher_order/add?domainKey=udinghuo&serviceCode=orderList")

// 卡片浏览态
window.open("https://dbox.yonyoucloud.com/mdf-node/meta/voucher/voucher_order/2086885027209543680?domainKey=udinghuo&serviceCode=orderList")
```

### 跳转报表

1. 获取报表链接（从报表界面获取 serviceCode）
2. `window.open('https://dbox.yonyoucloud.com/iuap-data-analytic/ucf-wh/analysis.html#/report/<serviceCode>?serviceCode=<serviceCode>')`

也可通过 `window.jDiwork.openService` 在工作台打开报表：

```js
let obj = {
  title: '报表',
  url: url,
  code: 'cpu_sourcing_pricedecision_card' + quotationid,
  type: '',
  serviceType: 'report',
  recordLogFlag: false,
};
window.jDiwork.openService('8a123e23-5fe5-4d26-b870-4b4cd99d8d1c', { serviceCode: '8a123e23-5fe5-4d26-b870-4b4cd99d8d1c' }, obj);
```

## 二、cb.loader.runCommandLine

```js
cb.loader.runCommandLine(
    'bill',
    {
      billtype: 'voucherlist',  // voucherlist列表 / voucher详情
      billno: 'pageRouteList',
      params: {
        id: '',                 // mode为edit/browse时必填
        // templateType: 'modal', // 弹窗方式
        mode: '',               // add / edit / browse
        domainKey: 'developplatform',
        query: { busiObj: 'pageRoute', serviceCode: '2110474997617131529' },
        carryParams: { new1: '111' },
        extraConfig: {
          _innerType: 'fadeRightToLeft', // 模态框位置
        }
      },
    },
    viewModel
);
```

## 三、window.jDiwork.openService

只能在工作台打开：

```js
// 打开详情新增态
let data = { billtype: 'voucher', billno: 'pageRoute', mode: 'add' };
window.jDiwork.openService('2110474997617131529', data);

// 打开详情浏览态
let data = { billtype: 'voucher', billno: 'pageRoute', billid: '2111355500281462785', mode: 'browse' };
window.jDiwork.openService('2110474997617131529', data);

// 打开列表
window.jDiwork.openService('2110474997617131529', {abc:1});

// 打开单张凭证
window.jDiwork.openService('newvoucher', { billid: voucherId }, { type: 1, code: voucherId });

// 打开多张凭证
window.jDiwork.openService('queryvoucher', { voucherids: "1904748228521230347,1904749551371157513" });
```
