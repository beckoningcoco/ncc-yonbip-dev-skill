# 取值科学计数法格式如何格式化

调用标准接口查询返回的结果中数值可能是科学计数法格式，需要自行转换。

```java
// 支持科学计数法和正常格式
private String getRealDecimal(Object moneny) {
    if (moneny == null) {
        return null;
    }
    DecimalFormat df = new DecimalFormat("0.##########"); // 保留所有小数位
    String result = df.format(moneny);
    return result;
}
```
