# 旗舰版中的两个用户档案

```sql
-- 用户 base.user.User
SELECT * FROM iuap_apcom_auth."user"

-- 友互通用户
SELECT * FROM iuap_apcom_auth.ba_user
```

- 用户表的 `yht_user_id` = 友互通用户表的 `id`
- 报销单的创建人参照是 `base.user.User`
- 调用 OpenAPI 时传的 `creator_code` 是 `base.user.User` 的 `code`
- 通过用户获取 token 时，使用的用户 id 是 `yht_user_id`（即 `ba_user` 的 `id`）
- 员工参照上的 `user_id` 也是 `iuap_apcom_auth.ba_user` 的 `id`
