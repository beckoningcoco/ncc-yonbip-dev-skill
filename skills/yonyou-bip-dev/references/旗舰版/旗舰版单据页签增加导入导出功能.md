# 旗舰版单据页签增加导入导出功能

## 1. 加按钮

在 UI 模板设计器中为页签添加导入和导出按钮。

## 2. 配置动作

### 导出配置

```
动作名称：分摊明细页签导出
命令：cmdFTMXTempExport
请求方式：openexportworkbench
方法：POST
扩展参数：{"supportUri": "earap.collection.CollectAllocationDetails", "exportType": "4" }
```

> 注意：将 `supportUri` 改成自己页签对应的实体 uri，`exportType` 不用变。

### 导入配置

```
动作名称：分摊明细页签导入
命令：cmdFTMXTempImport
请求方式：uploadandrender
方法：POST
请求地址：/bill/editImport
扩展参数：{"supportUri": "earap.collection.CollectAllocationDetails"}
```

> 命令、请求方式、请求地址固定不变，只修改扩展参数中的 uri。

## 3. 给按钮配置上动作

将创建好的动作绑定到按钮上。
