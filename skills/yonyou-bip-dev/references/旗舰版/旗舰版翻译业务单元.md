# 旗舰版翻译业务单元

```java
@Resource
private IOrgUnitQueryService orgUnitQueryService;

// 根据编码批量获取ID
String orgId = (String) reqBody.get("orgId");
List<String> idList = orgUnitQueryService.listIdsByCodes(InvocationInfoProxy.getTenantid(), Collections.singletonList(orgId), Collections.singletonList(1));
if (CollectionUtils.isNotEmpty(idList)) {
    reqBody.replace("orgId", idList.get(0));
}
```

```java
@Resource
private IOrgUnitQueryService orgUnitQueryService;

// 处理报文的 accentity 翻译会计主体
String accentity = (String) reqBody.get("accentity");
if (StringUtils.isNotBlank(accentity)) {
    ConditionDTO conditionDTO = new ConditionDTO();
    conditionDTO.andCodeIn(Collections.singletonList(accentity));
    List<OrgUnitDTO> queryList = orgUnitQueryService.getByCondition(InvocationInfoProxy.getTenantid(), conditionDTO);
    if (queryList == null || queryList.size() == 0) {
        throw new BusinessException("999", "传入的会计主体编码[" + accentity + "]在BIP系统中不存在，请检查！");
    }
    reqBody.put("accentity", queryList.get(0).getId());
}
```
