# 获取审批流信息

## 方式一：BpmUtils

```java
JSONObject workflowInfo = BpmUtils.getWorkflowInfo("RBSM", billnum, billId);
```

## 方式二：手动调用审批流 API

### 常量定义

```java
// 获取审批流信息
public final static String GET_WORKFLOW_INFO = "/iuap-apcom-workflownew/approve-component/api/v1/approve/load";
```

### 调用方法

```java
public static JSONObject getWorkflowInfo(String tenantAppSource, String busiObj, String billId) {
    Map<String, Object> param = new HashMap<>();
    param.put("businessKey", busiObj + "_" + billId);
    HttpResponse res =
            HttpUtil.createPost(EnvConfigUtil.getEnvConfig("domain.url") + BIPUrlConstant.GET_WORKFLOW_INFO)
                    .contentType("application/json;charset=UTF-8")
                    .body(JSONUtil.toJsonStr(param))
                    .header("Tenantappsource", tenantAppSource)
                    .timeout(30000)
                    .execute();
    if (res == null || res.getStatus() != 200) {
        throw new BusinessException("获取流程信息异常");
    }
    JSONObject info = JSONUtil.parseObj(res.body());
    if (MapUtils.isEmpty(info)) {
        throw new BusinessException("获取流程信息异常");
    }
    if (!info.containsKey("errcode") || info.getInt("errcode") != 0) {
        throw new BusinessException("获取流程信息异常：" + info.getStr("errmsg", "未知错误"));
    }
    return info.getJSONObject("data");
}
```
