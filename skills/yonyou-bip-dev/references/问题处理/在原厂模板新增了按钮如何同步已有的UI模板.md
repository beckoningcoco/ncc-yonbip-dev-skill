# 在原厂模板新增了按钮如何同步已有的UI模板

## 问题场景

在应用构建中对事项分录卡片做扩展，新增了"推送 sap 凭证"的按钮，结果只有原厂模板上有按钮，已经建好的 UI 模板没有这个按钮。

## 原因

事项分录没有交易类型特性，无法通过交易类型同步原厂按钮。

参考截图中的处理方式，需要通过其他途径同步 UI 模板。
