# 应收发票接口报错，非专岗环节不允许上传影像

## 场景

客开了一个应收发票接口：保存并自动提交，同时上传影像。现象是单据保存提交后立即上传影像报错"非专岗环节不允许上传影像"。

## 原因

单据配置了审批流，提交后需要等影像任务进入专岗环节（`ispostnode = true`）才能上传影像。

## 解决方案

循环查询 `img_task` 表，等待 `ispostnode` 变为 `true`：

```java
if (imagefiles != null && imagefiles.size() > 0) {
    int i = 0;
    while (i++ <= 10) {
        Thread.sleep(300);
        List<IBillDO> resList = (List) billQueryRepository.queryBySchema("sscimg.sscimg.img_task", querySchema2);
        if (resList != null && resList.size() > 0) {
            IBillDO iBillDO = resList.get(0);
            Boolean ispostnode = (Boolean) iBillDO.getAttrValue("ispostnode");
            // ispostnode 为 true 表示进入了专岗环节
            if (true == ispostnode) {
                break;
            }
        }
    }

    for (Map<String, Object> fileMap : imagefiles) {
        String imageUrl = (String) fileMap.get("imageUrl");
        String imageName = (String) fileMap.get("imageName");
        byte[] fileBytes = Base64Util.base64ToBytes(
            FileDownloadUtil.downloadFileWithInfo(imageUrl).getBase64()
        );
        cwgxSscImgService.uploadImg(id, orgCode, false, false, imageName, imageBase64, fileBytes);
    }
}
```
