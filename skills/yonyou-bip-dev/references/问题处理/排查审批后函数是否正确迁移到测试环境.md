# 排查某节点的动作配置的审批后函数是否正确迁移到测试环境

## 问题场景

收款单在审核动作扩展了一个函数，在 UAT 环境没有生效，怀疑没有迁移到 UAT。

## 排查 SQL

```sql
-- 查看扩展配置
SELECT * FROM "iuap_metadata_service"."uimeta_extension_tenant"
WHERE "biz_object" = 'collection'
  AND "extension_point_code" = 'afterAudit'
  AND dr = 0;

-- 查看扩展函数代码
SELECT * FROM "iuap_yonbuilder_service"."hpa_extcode"
WHERE id = (
  SELECT "script_id" FROM "iuap_metadata_service"."uimeta_extension_tenant"
  WHERE "biz_object" = 'collection'
    AND "extension_point_code" = 'afterAudit'
    AND dr = 0
);

-- 查看扩展函数内容
SELECT * FROM "iuap_yonbuilder_service"."hpa_extcode_content"
WHERE "fk_ext_code" = (
  SELECT "script_id" FROM "iuap_metadata_service"."uimeta_extension_tenant"
  WHERE "biz_object" = 'collection'
    AND "extension_point_code" = 'afterAudit'
    AND dr = 0
);
```
