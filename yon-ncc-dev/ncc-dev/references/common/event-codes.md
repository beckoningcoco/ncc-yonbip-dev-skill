# NCC 业务插件事件码对照表

业务插件（事件监听器）通过 `event.getEventType()` 获取事件类型码。

## 生命周期事件

| 事件码 | 说明 | 典型用途 |
|--------|------|----------|
| 1001 | 新增前 | 数据校验、默认值设置 |
| 1002 | 新增后 | 关联单据生成、回写 |
| 1003 | 修改前 | 数据校验、变更记录 |
| 1004 | 修改后 | 关联回写、同步更新 |
| 1005 | 删除前 | 删除校验、清理依赖 |
| 1006 | 删除后 | 清理关联回写 |
