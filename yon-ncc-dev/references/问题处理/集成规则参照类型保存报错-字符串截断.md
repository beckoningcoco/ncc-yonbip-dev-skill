# 集成规则配置-参照类型保存报错

## 环境

- **项目**: 水投
- **版本**: NCC 2207
- **包**: 2207 资产包

## 现象

配置集成规则（`pub_interuleitem` 表）时，字段类型选择"参照类型"，点击保存后报错字符串截断。

## 原因

`pub_interuleitem` 表体的 `vfieldtype`（字段类型）字段，建表时长度设置为 **20**。当选择参照类型时，参照的 ID 超过 20 位，写入时字符串被截断。

## 解决方案

达梦 SQL，将字段长度扩到 64：

```sql
ALTER TABLE pub_interuleitem MODIFY vfieldtype VARCHAR(64);
```
