# 旗舰版的日志如何在客开代码中记录？

- 使用以下代码

```
    Logger log = LoggerFactory.getLogger(PatSnapServiceImpl.class);
    log.error("sftp智慧芽文件已下载完成，准备处理！");
```



# 旗舰版客开代码中记录的日志如何查看，在哪里查看？或者说会被记录在哪里？

- 这里的日志，指的是开发人员在 客开的代码中，通过 Logger log = LoggerFactory.getLogger(PatSnapServiceImpl.class);去记录的日志，应该到哪里查看。
- 第一步，登录技术中台
- 第二步，微服务管理节点，找到客开的微服务/对应写记录日志代码的微服务
- 第三步，切换实例管理，找到YMS日志下载
- 第四步，找到/微服务编码/yms-application.log文件，这个就是记录你客开日志的地方。



# 关于记录日志的级别

- 首先推荐使用   log.error("sftp智慧芽文件已下载完成，准备处理！");   error级别去记录日志。因为BIP系统默认记录日志的级别是 error，低于error级别的都会被忽略掉！
- 如何修改系统的 日志记录级别？  
  - 登录技术中台--> 打开配置管理节点 ---> 切换到正确的环境---> 从左侧菜单找到你的客开微服务/你需要更改日志记录级别的微服务 ----> 切换到日志页签 ---->可以看到有个配置项叫做 日志级别，系统默认为 error。
  - 将日志级别 修改为其他，例如 info，debug， 然后点击  单服务配置发布，选择你的微服务，发布后，重启微服务即可生效！
