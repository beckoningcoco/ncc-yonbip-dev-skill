# 20个页面跳转代码片段

## 基础单据跳转

### 1. 跳转到指定单据（cb.loader.runCommandLine）

```javascript
var bizObj = viewModel.getAllData();
let request = {
    billtype: 'Voucher',       // Voucher=卡片页, VoucherList=列表页
    billno: 'efd1d091',        // 目标单据号（从浏览器地址栏Voucher后获取）
    params: {
        mode: 'add',           // add/edit/browse
        readOnly: false,
        data: bizObj           // 传递的业务数据
    }
};
cb.loader.runCommandLine('bill', request, viewModel);
```

### 2. 动态模式跳转

```javascript
var req = viewModel.getParams();
var mode = req.mode || 'add';
let billNoSet = viewModel.get('billNoSet').getValue();
let billNoList = JSON.parse(billNoSet);
let request = {
    billtype: 'Voucher', billno: billNoList.decision,
    params: { mode: mode, readOnly: false, data: viewModel.getAllData() }
};
cb.loader.runCommandLine('bill', request, viewModel);
```

### 3. 浏览模式查看详情（传 ID）

```javascript
var grid = viewModel.getGridModel();
var row = grid.getFocusedRowIndex();
var data = grid.getRow(row);
let request = {
    billtype: 'Voucher', billno: 'a295205e',
    params: { mode: 'browse', readOnly: true, id: data.id }
};
cb.loader.runCommandLine('bill', request, viewModel);
```

## 条件跳转

### 4. 根据业务条件跳转不同页面

```javascript
var ecoSituation = viewModel.get("ecoSituation").getValue();
var billNo = '';
if ("44" == ecoSituation) { billNo = '3c1fd2ed'; }
else if ("45" == ecoSituation) {
    var aprsBasis = viewModel.get("aprsBasisHide").getValue();
    billNo = ("1" == aprsBasis) ? '954bb989' : 'a5c6e484';
}
if (billNo) { cb.loader.runCommandLine('bill', request, viewModel); }
else { cb.utils.alert('目标页面不存在'); }
```

## 模态框操作

### 5. 关闭模态框并返回数据

```javascript
viewModel.get('button28hb').on('click', function (data) {
    var selectedRows = viewModel.getGridModel().getSelectedRows();
    if (selectedRows.length === 0) { cb.utils.alert('请选择一条数据'); return; }
    viewModel.communication({ type: 'return', payload: { type: 'confirm', data: selectedRows[0] } });
});
```

### 6. 关闭模态框不返回数据

```javascript
viewModel.communication({ type: 'modal', payload: { data: false } });
```

### 7. 手动返回上个页面

```javascript
viewModel.get('button21og').on('click', function () {
    cb.utils.alert('返回成功');
    viewModel.communication({ type: 'return' });
});
```

## 接收父页面参数

### 13. 接收父页面传递的参数

```javascript
viewModel.on('afterMount', function () {
    const salary_amount = viewModel.getParams().salary_amount || viewModel.getParams().query.salary_amount;
    const filtervm = viewModel.getCache('FilterViewModel');
    filtervm.on('afterInit', function () {
        filtervm.get('salary_amount').getFromModel().setValue(salary_amount);
```

## 跳转参数说明

### 完整参数结构

```javascript
let request = {
    billtype: 'Voucher',       // Voucher(卡片页) / VoucherList(列表页)
    billno: '0b50d1b4',        // 单据号
    params: {
        mode: 'add',           // add(新增) / edit(编辑) / browse(浏览)
        readOnly: false,       // 是否只读
        id: '123',             // 记录ID（编辑/浏览模式需要）
        data: { field1: value1, field2: value2 },  // 业务数据
        domainKey: 'scapoe'    // 领域编码
    }
};
```

## 关键 API

| API | 说明 |
|-----|------|
| `cb.loader.runCommandLine('bill', request, vm)` | 跳转单据/列表页 |
| `viewModel.communication({ type: 'return', payload })` | 关闭模态框并返回数据 |
| `viewModel.communication({ type: 'modal', payload: { data: false } })` | 关闭模态框不返回 |
| `jDiwork.openService(code, dt, param)` | 新标签页打开 |
| `window.open(url, '_blank')` | 跳转外部系统 |
| `rootStore.pageStore.push({...})` | rootStore 跳转 |
| `viewModel.getParams()` | 获取页面参数（含父页面传参） |
