# 异步执行代码

## 低版本

```java
import java.util.concurrent.ExecutorService;
private final ExecutorService ymsExecutor = YmsExecutors.getYmsExecutor();

ymsExecutor.execute(() -> {
    // 异步执行，防止前台超时
});
```

## 高版本

```java
private final IYmsExecutorService ymsExecutor = YmsExecutors.getYmsExecutor();
```

## 使用示例

```java
if (id != null) {
    final Long needId = id;
    ymsExecutor.execute(() -> {
        try {
            yonbipUtil.batchAuditSaleOrder(Arrays.asList(needId));
            updateAuditorBysql(needId, (Long) reqBody.get("creatorId"), (String) reqBody.get("creator"));
        } catch (Exception e) {
            // 审核失败不报错，记录日志
            String org = saleOrder.get("org") == null ? "" : saleOrder.get("org").toString();
            Long customerId = (Long) saleOrder.get("customer");
            String dept = saleOrder.get("department") == null ? null : saleOrder.get("department").toString();
            String billno = saleOrder.get("code") == null ? "" : saleOrder.get("code").toString();
            String billType = "销售订单";
            yonbipUtil.makeErrorLogsForInterface(org, customerId, dept, billno, billType, e.getMessage());
        }
    });
}
```
