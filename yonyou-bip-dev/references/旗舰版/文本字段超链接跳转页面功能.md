# 文本字段超链接跳转页面功能

## 需求描述

点击单据页签上的"凭证链接"文本字段，动态跳转到对应的 SAP 凭证详情卡片界面（跳转地址非固定）。

## 实现方式

### 1. 字段类型保持为"文本"（不要设置成超链接类型）

超链接类型只能配置固定地址，不符合动态跳转需求。

### 2. 设置字段属性，将文本转换成超链接

```js
// 数据加载后，设置页签链接字段的 joinQuery 属性为 true
viewModel.on('afterLoadData', function () {
  var sapModel = viewModel.get('PubPrePaySapVoucherList');
  if (sapModel) {
    // 第一个参数是字段名，第二个是固定的属性名称，第三个设置为 true
    sapModel.setColumnState('LJ', 'bJointQuery', true);
    sapModel.setColumnState('LJ', 'enableJointQuery', true);
  }
});
```

### 3. 监听超链点击事件，进行页面跳转

```js
viewModel.getGridModel('PubPrePaySapVoucherList').on('beforeCellJointQuery', (args) => {
  if (args.cellName && args.cellName == 'LJ') {
    let LJid = args.row.LJ; // LJ 字段存的是目标单据的 id

    // cb.loader.runCommandLine 方式
    // cb.loader.runCommandLine('bill', {
    //   billtype: 'voucher',
    //   billno: 'SapVoucherDetails',
    //   domainKey: 'c-kk-lzlj-cwgx',
    //   params: { id: LJid, mode: 'browse' },
    // }, viewModel);

    // jDiwork.openService 方式（推荐，可在新页签内打开）
    jDiwork.openService(
      'SapVoucherDetailsList',
      {
        billtype: 'voucher',
        billno: 'SapVoucherDetails',
      },
      {
        title: 'Sap凭证详细信息',
        data: { id: LJid, mode: 'browse', readOnly: true },
        code: LJid,
      }
    );
    return false;
  }
});
```
