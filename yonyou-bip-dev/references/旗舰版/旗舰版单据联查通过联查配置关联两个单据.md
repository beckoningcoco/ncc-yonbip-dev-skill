# 旗舰版单据联查，通过联查配置关联两个单据

## 场景

收款单和应收发票本身没有上下游关系，通过联查配置让两个单据关联起来，在全局联查中可见。

## 联查配置步骤

1. 进入**联查配置**节点
2. 配置关联关系

## 关键参数获取

- **下游单据号**：从业务对象上或 UI 模板里获取
- **所属应用**：从应用构建中找到，或直接在联查配置节点搜索单据查看预制配置中的应用编码
