# 旗舰版富文本转成普通字符串

## 目标

将如下字符串：

```js
const targetContent = '<div forprinttag="true" class="mdf_rich_text">n<div>成交信息</div>n</div>';
```

转换为纯文本：

```
成交信息
```

------

## 解决方案：使用纯 JavaScript 正则 + 字符串处理（无需 document）

```js
function htmlToText(html) {
  if (!html) return '';

  return html
    .replace(/<style[^>]*>[\s\S]*?<\/style>/gi, '')
    .replace(/<script[^>]*>[\s\S]*?<\/script>/gi, '')
    .replace(/<[^>]+>/g, '')
    .replace(/&nbsp;/gi, ' ')
    .replace(/&amp;/gi, '&')
    .replace(/&lt;/gi, '<')
    .replace(/&gt;/gi, '>')
    .replace(/n/g, '')
    .replace(/\s+/g, ' ')
    .trim();
}

// 使用示例
const targetContent = '<div forprinttag="true" class="mdf_rich_text">n<div>成交信息</div>n</div>';
const text = htmlToText(targetContent);
console.log(text); // 输出: 成交信息
```
