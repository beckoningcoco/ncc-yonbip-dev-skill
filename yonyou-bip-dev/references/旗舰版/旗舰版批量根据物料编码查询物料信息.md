# 旗舰版批量根据物料编码查询物料信息

```java
/**
 * 批量根据物料编码查询物料的单位ID
 * @param productCodes 物料编码列表
 * @return Map<productCode, unitId>
 */
public Map<String, String> batchGetProductUnitId(List<String> productCodes) throws Exception {
    Map<String, String> result = new HashMap<>();
    if (productCodes == null || productCodes.isEmpty()) {
        return result;
    }
    ProductQryDTO productQryDTO = new ProductQryDTO();
    productQryDTO.setProductCodeList(productCodes);

    List<ProductDTO> productDTOList = productService.listProductInfo(productQryDTO,
            new String[] { "id", "code", "unitId" });
    productDTOList.forEach(product -> {
        if (product != null && product.getUnitId() != null) {
            result.put(product.getCode().toString(), product.getUnitId().toString());
        }
    });
    return result;
}
```

- 注入 `@Autowired private IProductServiceV2 productService;`
- `listProductInfo` 的第二个参数指定需要返回的字段数组
