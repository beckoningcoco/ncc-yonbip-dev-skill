# 旗舰版查询物料业务信息上的自定义特征值

```java
@SuppressWarnings("all")
public Map<String, Object> checkMaterialIsZK(String[] productCodes, String orgId) {

    if (productCodes == null || productCodes.length == 0) {
        throw new BusinessException("999", "productCodes 不能为空");
    }

    // 查询 global00 业务单元对应的 id
    String global00Id = "";
    QuerySchema querySchema = QuerySchema.create()
            .addSelect("id", "code", "name")
            .appendQueryCondition(QueryCondition.name("enable").eq(1))
            .appendQueryCondition(QueryCondition.name("dr").eq(0))
            .appendQueryCondition(QueryCondition.name("code").eq("global00"))
            .appendQueryCondition(QueryCondition.name("ytenant").eq(InvocationInfoProxy.getTenantid()));
    List<IBillDO> docList = (List) billQueryRepository.queryBySchema("org.func.BaseOrg", querySchema);
    if (docList != null && docList.size() > 0) {
        global00Id = docList.get(0).getAttrValue("id") == null ? null
                : docList.get(0).getAttrValue("id").toString();
    }

    // 查询所有物料的管理组织（orgId 代表物料管理组织，可能是企业级也可能是组织级）
    Map<String, Object> orgUnitMap = new HashMap<>();
    querySchema = QuerySchema.create()
            .addSelect("id", "code", "name", "orgId")
            .appendQueryCondition(QueryCondition.name("code").in(productCodes))
            .appendQueryCondition(QueryCondition.name("ytenant").eq(InvocationInfoProxy.getTenantid()));
    docList = (List) billQueryRepository.queryBySchema("pc.product.Product", querySchema);
    if (docList != null && docList.size() > 0) {
        for (IBillDO billDO : docList) {
            String code = billDO.getAttrValue("code") == null ? "" : billDO.getAttrValue("code").toString();
            String orgglId = billDO.getAttrValue("orgId") == null ? "" : billDO.getAttrValue("orgId").toString();
            orgUnitMap.put(code, orgglId);
        }
    }

    // 按组织查询物料业务信息（产品明细）
    QueryConditionGroup condg = new QueryConditionGroup(ConditionOperator.or);
    condg.addCondition(QueryCondition.name("productDetail.orgId").eq(global00Id));
    condg.addCondition(QueryCondition.name("productDetail.orgId").eq(orgId));
    condg.addCondition(QueryCondition.name("productDetail.orgId").in(orgUnitMap.values()));
    querySchema = QuerySchema.create()
            .addSelect("id", "code", "name", "productDetail.orgId", "productDetail.*")
            .appendQueryCondition(QueryCondition.name("code").in(productCodes))
            .appendQueryCondition(condg)
            .appendQueryCondition(QueryCondition.name("ytenant").eq(InvocationInfoProxy.getTenantid()));
    docList = (List) billQueryRepository.queryBySchema("pc.product.Product", querySchema);

    // 从 extendData 中获取自定义特征值（如 MGPM07）
    Map<String, Object> klmap = new HashMap<>();
    if (CollectionUtils.isNotEmpty(docList)) {
        for (IBillDO billDO : docList) {
            BigDecimal zkl = BigDecimal.ZERO;
            String materialCode = billDO.getAttrValue("code") == null ? "" : billDO.getAttrValue("code").toString();
            Map<String, Object> extendData = (Map<String, Object>) billDO.getAttrValue("extendData");
            if (extendData != null) {
                Map<String, Object> def = (Map<String, Object>) extendData
                        .get("productDetail_productExtendCharacterDef");
                if (def != null) {
                    zkl = def.get("MGPM07") == null ? BigDecimal.ZERO
                            : new BigDecimal(def.get("MGPM07").toString());
                }
            }
            String org_id = billDO.getAttrValue("productDetail_orgId") == null ? ""
                    : billDO.getAttrValue("productDetail_orgId").toString();
            if (zkl.compareTo(BigDecimal.ZERO) > 0) {
                String glorg = orgUnitMap.get(materialCode) == null ? "" : orgUnitMap.get(materialCode).toString();
                if (glorg.equals(org_id) && klmap.get(materialCode) == null) {
                    klmap.put(materialCode, zkl);
                } else if (orgId.equals(org_id)) {
                    klmap.put(materialCode, zkl);
                }
            }
        }
    }
    return klmap;
}
```

## 关键点

- 自定义特征值从 `extendData.productDetail_productExtendCharacterDef` 中读取
- 物料可能是企业级（global00）也可能是组织级，查询时需 OR 三种 orgId 条件
- 取值优先级：物料自身管理组织 > 当前组织 > 企业级
