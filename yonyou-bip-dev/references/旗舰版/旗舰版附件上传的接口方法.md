# 旗舰版附件上传的接口方法

## 方式一：对方传 Base64 编码

```java
//报文里传附件的base64编码
List<Map<String, Object>> attachment = (List<Map<String, Object>>) params.remove("attachment");
if (attachment != null && attachment.size() > 0) {
    String bussnessid = upDateAttachment(attachment);
    params.put("attachment", bussnessid);

```

```java
@Resource
private CooperationFileUploadService cooperationFileUploadService;

private String upDateAttachment(List<Map<String, Object>> attachmentList) throws Exception {
    String businessType = "c-kk-lzlj-cwgx+mdf"; //这个不是领域编码，得去前台上传个文件从控制台处获取
    String businessId = null;
    for (Map<String, Object> base64file : attachmentList) {
        if (businessId == null) {
            businessId = IdUtil.simpleUUID();
        }
        String content = (String) base64file.get("content");
        byte[] fileBytes = decodeBase64ToBytes(content);
        cooperationFileUploadService.uploadFile(businessType, businessId, fileBytes, (String) base64file.get("fileName"), null);
    }
    return businessId;
}

private static byte[] decodeBase64ToBytes(String base64Str) {
    String pureBase64 = extractBase64(base64Str);
    return Base64.getDecoder().decode(pureBase64);
}

private static String extractBase64(String base64Str) {
    if (base64Str != null && base64Str.startsWith("data:")) {
        int commaIndex = base64Str.indexOf(',');
        if (commaIndex != -1) {
            return base64Str.substring(commaIndex + 1);
        }
    }
    return base64Str;
}
```

## 方式二：对方传下载文件地址

```java
private void handleAttachFiles(Map<String, Object> data) {
    List<Map<String, Object>> attachList = (List<Map<String, Object>>) data.get("attachs");
    if (attachList != null && attachList.size() > 0) {
        String businessType = "ZNBZ";
        String businessId = null;
        for (Map<String, Object> attachMap : attachList) {
            if (businessId == null) {
                businessId = IdUtil.simpleUUID();
            }
            String fileUrl = (String) attachMap.get("attachUrl");
            String fileName = (String) attachMap.get("attachName");
            FileDownloadUtil.FileDownloadResult result = null;
            try {
                result = FileDownloadUtil.downloadFileWithInfo(fileUrl);
            } catch (Exception e) {
                throw new BusinessException("999", e.getMessage());
            }
            String imageBase64 = result.getBase64();
            byte[] fileBytes = Base64Util.base64ToBytes(imageBase64);
            cooperationFileUploadService.uploadFile(businessType, businessId, fileBytes, fileName, null);
        }
        data.put("vattachmentass", businessId);
    }
}
```

## FileDownloadUtil

完整代码参见 [[旗舰版影像上传]]，核心功能：
- `downloadFileWithInfo(fileUrl)` — 下载文件返回文件名 + Base64
- `downloadFileAsFile(fileUrl)` — 下载文件返回 File 对象
- 支持图片压缩（按文件大小分级压缩）
- 支持 HTTP→HTTPS 自动重试

## Base64Util

完整代码参见 [[旗舰版影像上传]]，核心功能：
- `base64ToBytes(base64Str)` — Base64 转字节数组
- `getFileSuffix(base64Str, defaultExt)` — 从 Base64 提取文件后缀（支持 MIME 和 Magic Number 判断）
