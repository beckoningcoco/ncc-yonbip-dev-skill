# 旗舰版页面脚本关闭弹窗

## 关闭弹窗不刷新父页面

```js
viewModel.get('button5ri') && viewModel.get('button5ri').on('click', function (data) {
    //取消--onClick
    let parentGridModel = viewModel.getCache('parentViewModel');
    if (!parentGridModel) {
      cb.utils.alert('按钮只允许在父界面下使用！', 'error');
      return;
    }
    //关闭弹窗不刷新父页面
    viewModel.communication({ type: 'return', payload: { cParameter: JSON.stringify({ refresh: false }) } });
  });
```

## 关闭弹窗（默认自动刷新父页面）

```js
//关闭弹窗
viewModel.communication({ type: 'return' }); //关闭默认自动刷新父页面
```
