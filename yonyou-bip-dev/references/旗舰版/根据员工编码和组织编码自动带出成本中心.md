# 根据员工编码和组织编码自动带出成本中心

```java
// 成本中心
String cfinacecostcenter = dto.getCfinacecostcenter();
if (StringUtils.isNotBlank(cfinacecostcenter)) {
    BizObject costCenter = BaseInfoUtils.queryByCode(MetaDataEnum.CostCenter, "id", cfinacecostcenter);
    if (MapUtils.isEmpty(costCenter)) {
        throw new BusinessException("未获取到成本中心，编码：" + cfinacecostcenter);
    }
    dto.setCfinacecostcenter(costCenter.getId().toString());
} else {
    String costCenterByStaffCode =
            baseDocService.getCostCenterByStaffCode(dto.getPk_handlepsn(), dto.getCfinaceorg(), null);
    dto.setCfinacecostcenter(costCenterByStaffCode);
}
```

## getCostCenterByStaffCode 实现

优先获取默认成本中心，若无默认则获取非默认的第一个：

```java
@Override
public String getCostCenterByStaffCode(String staffCode, String orgCode, Boolean isDefault) {
    HashMap<String, Object> fieldValue = new HashMap<>();
    if (isDefault != null) {
        fieldValue.put("isDefault", isDefault);
    }
    fieldValue.put("staff.code", staffCode);
    fieldValue.put("baseOrg.code", orgCode);
    List<Map<String, Object>> userCostCenterList = BaseInfoUtils.queryListByMap(
        MetaDataEnum.userCostCenter, "id,isDefault,costCenter.code", false, fieldValue
    );
    if (CollectionUtils.isEmpty(userCostCenterList)) {
        throw new RuntimeException("未获取到员工成本中心，员工编码：" + staffCode + "，公司编码：" + orgCode);
    }
    // 优先获取默认成本中心
    if (isDefault == null || isDefault) {
        for (Map<String, Object> map : userCostCenterList) {
            if (map.get("isDefault") != null && (Boolean) map.get("isDefault") && map.get("costCenter_code") != null) {
                return map.get("costCenter_code").toString();
            }
        }
    }
    // 获取非默认成本中心
    if (isDefault == null || !isDefault) {
        for (Map<String, Object> map : userCostCenterList) {
            if (map.get("costCenter_code") != null) {
                return map.get("costCenter_code").toString();
            }
        }
    }
    throw new BusinessException("未获取到员工成本中心，员工编码：" + staffCode + "，公司编码：" + orgCode);
}
```

## BaseInfoUtils.queryListByMap 封装

```java
public static List<Map<String, Object>> queryListByMap(
    MetaDataEnum metaDataEnum, String fields, boolean appendDr,
    HashMap<String, Object> conditionFieldValue
) {
    if (ObjectUtils.isEmpty(conditionFieldValue)) {
        return Collections.emptyList();
    }
    QuerySchema querySchema = QuerySchema.create();
    querySchema.addSelect(fields);
    QueryConditionGroup conditionGroup = new QueryConditionGroup(ConditionOperator.and);
    for (String key : conditionFieldValue.keySet()) {
        conditionGroup.appendCondition(QueryCondition.name(key).eq(conditionFieldValue.get(key)));
    }
    if (appendDr) {
        conditionGroup.appendCondition(QueryCondition.name("dr").eq(0));
    }
    querySchema.addCondition(conditionGroup);
    return executeQuery(metaDataEnum, querySchema);
}
```
