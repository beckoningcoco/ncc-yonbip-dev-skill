# 扩展规则链规则写法

## 规则链扩展配置

- 如何配置规则链？如何在规则链中配置一个我们自己的客开的规则？如何新增规则？
  - 进入应用构建 --> 找到对应模块，对应的页面 -->  页面建模 --->找到命令管理按钮 ---> 找到需要添加规则链的命令（一般要切换到动作页签） ---->新增规则即可。
  - 注意，规则的命名 ，若规则类上没有  @Component("xxxxx") 注解，则命名应该为  类名，首字母小写 ； 若有   @Component("xxxxx") ，说明指定了类名，则命名要填xxxxx，和注解内容保持一致。
