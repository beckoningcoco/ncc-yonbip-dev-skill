# 设置内部交易规则

客开调拨订单时，如果对方系统不传单价、含税金额等字段，可在报文中设置 `autoGetPrice` 让系统自动计算价格：

```java
reqTransferBillMap.put("autoGetPrice", true);
```

设置了自动取价后，需要以下配置：

## 1. 配置内部交易规则

在**内部交易规则**节点配置调拨关系的价格规则。

## 2. 配置定价中心调价单

在**定价中心 → 调价单**中设置物料的价格：

- 新建调价单，选择对应的物料和组织
- 设置单价、含税单价等
- 审核后生效

## 配置节点

- 内部交易规则
- 定价中心 → 调价单
