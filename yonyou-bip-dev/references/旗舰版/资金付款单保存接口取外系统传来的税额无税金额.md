# 资金付款单保存接口取外系统传来的税额无税金额

旗舰版资金付款单保存时，系统会根据传入的**含税金额** `oriSum` 和**税种** `taxCateregory_code` 自动计算税额和无税金额。

如果想取外系统传入的税额和无税金额（而非让系统自动计算），则需要同时传入以下四个字段：

- **税额** (`tax`)
- **税率** (`taxRate`)
- **无税金额** (`noTaxAmount`)
- **含税金额** (`oriSum`)

此时即使传了税种也不会触发自动计算，系统会直接使用传入的值。
