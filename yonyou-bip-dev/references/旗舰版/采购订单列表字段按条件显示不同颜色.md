# 采购订单列表字段按条件显示不同颜色

## 问题场景

客户想要在采购订单列表界面将"是否付款申请"字段按值显示不同颜色（类似"单据状态"）。

## 脚本

```js
viewModel.getGridModel().on('afterSetDataSource', (args) => {
  if (viewModel.get('sumSwitch').getValue() == true) {
    let rows = viewModel.getGridModel().getRows();
    rows.forEach((item, index) => {
      if (item.formula_userDefine_2077008767987744771 == '无付款申请') {
        viewModel.getGridModel().setCellState(index, 'formula_userDefine_2077008767987744771', 'style', { color: 'green' });
      } else {
        viewModel.getGridModel().setCellState(index, 'formula_userDefine_2077008767987744771', 'style', { color: 'red' });
      }
    });
  }
});
```
