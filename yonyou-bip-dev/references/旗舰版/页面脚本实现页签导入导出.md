# 页面脚本实现页签导入导出

## 导入按钮页面脚本

核心流程：
1. 动态加载 xlsx.js 库
2. 创建隐藏文件选择框，触发 Excel 文件选择
3. 读取 Excel 文件，解析工作表数据（跳过表头行）
4. 收集参照字段的编码，调用后端 API 批量翻译为 ID
5. 将翻译后的数据插入 gridModel 表格

完整脚本见原文（代码较长，包含字段映射配置、档案类型映射、领域映射、批量查询、错误收集等）。

关键配置结构：

```js
// Excel字段编码 -> 系统字段名
const TEMPLATE_FIELD_MAP = {
  id: 'id',
  customer_name: 'customer',
  supplier_name: 'supplier',
  // ...
};

// 引用字段的档案类型映射
const REFERENCE_ARCHIVE_TYPE = {
  businessType: 'bd.expenseitem.ExpenseItem',
  costCenter: 'bd.costcenter.CostCenter',
  // ...
};

// 档案对应领域配置
const REFERENCE_ARCHIVE_DOMAIN = {
  'bd.customerdoc_MYHB.MYHB': 'ucfbasedoc',
  // ...
};
```

## 后端 API 函数

用于根据档案类型和编码批量查询 ID：

```js
let AbstractAPIHandler = require('AbstractAPIHandler');
class MyAPIHandler extends AbstractAPIHandler {
  execute(request) {
    const { archiveType, codes, domain } = request;
    let nameStr = codes.map(name => "'" + name.replace(/'/g, "''") + "'").join(',');
    let result = [];
    switch (archiveType) {
      case 'bd.expenseitem.ExpenseItem':
        result = ObjectStore.queryByYonQL(`select id, code, name from bd.expenseitem.ExpenseItem where name in(${nameStr})`, domain);
        break;
      case 'bd.costcenter.CostCenter':
        result = ObjectStore.queryByYonQL(`select id, code, name from bd.costcenter.CostCenter where name in(${nameStr})`, domain);
        break;
      // ... 更多档案类型
      case 'aa.vendor.Vendor':
        result = ObjectStore.queryByYonQL(`select id, code, name from aa.vendor.Vendor where name in(${nameStr})`, domain);
        break;
    }
    return { code: 200, message: '查询成功', data: result };
  }
}
exports({ entryPoint: MyAPIHandler });
```

## 页面设计

- **导入按钮**：不需要引用动作，由页面脚本监听 click 事件
- **导出按钮**：引用模板下载动作
