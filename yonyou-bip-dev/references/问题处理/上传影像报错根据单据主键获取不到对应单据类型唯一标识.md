# 上传影像报错，根据单据主键获取不到对应单据类型唯一标识

## 问题

报错原因：根据单据主键没有查询到单据影像任务。

```sql
SELECT * FROM ssc_img.img_task WHERE billid = '2427993666065793032'
```

## 排查

如果任务已经被创建但仍然报错，检查调用 OpenAPI 时使用的 `appKey` 和 `appSecret` 是否正确（是否用了开发环境的而非 UAT 环境的）。更换后即可解决。
