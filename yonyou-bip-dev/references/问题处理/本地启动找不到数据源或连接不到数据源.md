# 本地启动找不到数据源或连接不到数据源

## 现象

本地启动项目时报错：
- 找不到数据源或连接不到数据源
- `xxxConenntTimeOut`（工程名带括号时）

## 排查与解决

### 1. 检查 yms-home 文件夹

项目启动时会生成 `yms-home` 文件夹。检查 `yms_middleware.json` 文件中的两个配置项：**Redis** 和 **URL**。

这两个配置项的 IP 可能不正确，需要找运维确认正确的 IP 后替换。

### 2. 修改不生效问题

注意：**在原文件上直接修改可能不生效**（重启时会自动覆盖），需要将对应的文件转移到相应的文件夹后再修改。

### 3. Git 工程名不能带括号

如果是从 Git 拉取的代码，**工程名不能带括号**，否则本地启动时会报 `xxxConenntTimeOut` 错误。
