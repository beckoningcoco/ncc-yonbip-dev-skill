# 没有费用项目的权限，如何分配

## 报错

```json
{
  "code": "1021000999999",
  "message": "[没有费用项目：修理费 的权限，请重新选择费用项目]"
}
```

## 解决办法

从费用项目相关节点，分配费用项目权限即可。
