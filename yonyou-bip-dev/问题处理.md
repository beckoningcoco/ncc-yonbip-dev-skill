## Maven 编译报错 NoSuchFieldError: JCTree$JCImport.qualid

- JDK 版本与 Lombok 不兼容，Lombok 通过反射访问 `com.sun.tools.javac.tree.JCTree$JCImport.qualid` 时找不到字段，通常是 JDK 小版本太新（如 Zulu 8u452）导致内部 API 变更。

参考`references/问题处理/Maven编译Lombok-NoSuchFieldError.md`
